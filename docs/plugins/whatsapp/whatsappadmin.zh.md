# 行政管理-WhatsApp模块

## 1. 现役人员
**何为App:** 
通过启用此插件,用户将能够:
- 通过WhatsApp访问并发送消息.
- 为自己的企业配置自动响应的聊天机.

---

## 2. 信贷
- 用户有一个 **WhatsApp 的独立钱包余额**。 。 。 。
- 管理员可以查看信用余额 **短消息** 财务报告和审定财务报表 **怎样?** 分别。

**添加信用的步骤 :**
1. 点击 **添加新内容** 将贷记入用户余额。
2. 选择 **怎样?** 从服务列表。
3. 输入 a **付款参考**。 。 。 。
4. 输入 **金额**。 。 。 。
5. 点击 **添加资信** 以填充用户的余额。

![WhatsApp Admin Credits](images/whatsappadmin1.png)

---

## 3. 用户比率计划
在WhatsApp区,你可以指定 **费率计划** 用于用户。

- 从下放菜单中选择速率计划 。
- 所有速率计划均在 **计费** 区域将在此显示供选择。

---

### 速率计划
类似于 MT 消息速率计划,管理员必须定义 **特定比率计划** 管理账单。

**可用动作 :**
1. **编辑** - 更改费率计划名称或启用/失效计划。
2. **视图** - 查看和修改利率计划中配置的所有价格。 更新将适用于所有指定的用户。
3. **复制** ——复制现有收费计划并取出新名称.
4. **删除** - 永久取消收费计划。 *(无法撤销)*

![WhatsApp Admin Rate Plans](images/whatsappadmin2.png)

## 制定新费率计划

**步骤1:** 
- 输入 **友好利率计划名称**。 。 。 。
- 选择 **活动/ 不活动** 状态。
- 点击 **继续**。 。 。 。

![WhatsApp Admin Step 1](images/whatsappadmin3.png)

---

**步骤2:** 
- 确定利率计划的价格。
- 为用户配置计费细节 。

![WhatsApp Admin Step 2](images/whatsappadmin4.png)

**字段 :**
- **国家代码** - 选择适用的国家。
- **价格** - META每条电文收取的金额。
- **销售价格** – 每条消息向用户收取的价格.
- **平台充电** - 使用平台的可选收费。

---

### 使用大小写

**案例1:** 
- 与META的账单 由管理员处理。 
- 不收取站台费. 
- **扣除的贷项** 总计 *销售价格*。 。 。 。

**案例2:** 
- 与META的帐单由用户处理. 
- 只收取站台费. 
- **销售价格** 保持为 <span data-ph="0"></span>。 。 。 。

**案例3:** 
- 管理人员既收取销售价格,又收取平台费。 
- 扣除的贷项 **销售价格+平台收费**。 。 。 。

---

**步骤3:** 
- 评议 **费率计划摘要**。 。 。 。
- 确保所有细节都正确。
- 点击 **保存** 最终敲定。
