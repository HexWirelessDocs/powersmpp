# 行政管理-WhatsApp模組

## 1. 現役人員
**何為App:** 
透過啟用此外掛,使用者將能夠:
- 透過WhatsApp訪問併發送訊息.
- 為自己的企業配置自動響應的聊天機.

---

## 2. 信貸
- 使用者有一個 **WhatsApp 的獨立錢包餘額**。 。 。 。
- 管理員可以檢視信用餘額 **短訊息** 財務報告和審定財務報表 **怎樣?** 分別。

**新增信用的步驟 :**
1. 點選 **新增新內容** 將貸記入使用者餘額。
2. 選擇 **怎樣?** 從服務列表。
3. 輸入 a **付款參考**。 。 。 。
4. 輸入 **金額**。 。 。 。
5. 點選 **新增資信** 以填充使用者的餘額。

![WhatsApp Admin Credits](images/whatsappadmin1.png)

---

## 3. 使用者比率計劃
在WhatsApp區,你可以指定 **費率計劃** 用於使用者。

- 從下放選單中選擇速率計劃 。
- 所有速率計劃均在 **計費** 區域將在此顯示供選擇。

---

### 速率計劃
類似於 MT 訊息速率計劃,管理員必須定義 **特定比率計劃** 管理賬單。

**可用動作 :**
1. **編輯** - 更改費率計劃名稱或啟用/失效計劃。
2. **檢視** - 檢視和修改利率計劃中配置的所有價格。 更新將適用於所有指定的使用者。
3. **複製** ——複製現有收費計劃並取出新名稱.
4. **刪除** - 永久取消收費計劃。 *(無法撤銷)*

![WhatsApp Admin Rate Plans](images/whatsappadmin2.png)

---

## 制定新費率計劃

**步驟1:** 
- 輸入 **友好利率計劃名稱**。 。 。 。
- 選擇 **活動/ 不活動** 狀態。
- 點選 **繼續**。 。 。 。

![WhatsApp Admin Step 1](images/whatsappadmin3.png)

---

**步驟2:** 
- 確定利率計劃的價格。
- 為使用者配置計費細節 。

![WhatsApp Admin Step 2](images/whatsappadmin4.png)

**欄位 :**
- **國家程式碼** - 選擇適用的國家。
- **價格** - META每條電文收取的金額。
- **銷售價格** – 每條訊息向用戶收取的價格.
- **平臺充電** - 使用平臺的可選收費。

---

### 使用大小寫

**案例1:** 
- 與META的賬單 由管理員處理。 
- 不收取站臺費. 
- **扣除的貸項** 總計 *銷售價格*。 。 。 。

**案例2:** 
- 與META的帳單由使用者處理. 
- 只收取站臺費. 
- **銷售價格** 保持為 <span data-ph="0"></span>。 。 。 。

**案例3:** 
- 管理人員既收取銷售價格,又收取平臺費。 
- 扣除的貸項 **銷售價格+平臺收費**。 。 。 。

---

**步驟3:** 
- 評議 **費率計劃摘要**。 。 。 。
- 確保所有細節都正確。
- 點選 **儲存** 最終敲定。
