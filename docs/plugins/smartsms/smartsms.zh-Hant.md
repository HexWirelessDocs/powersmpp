
# 智慧短訊息

## 智慧短訊息使用者指南

### 導言
歡迎光臨 **智慧短訊息**,Smart Message的最終解決方案,它不僅豐富了與客戶的移動接觸,而且賦予了您最前沿的資料分析能力. 
這個全面的指南將引導您瀏覽智慧短訊息的特性和功能,確保您充分利用這個強大的工具.

---

## 智慧連結

### 帶有跟蹤功能的縮短 URL
這個 **智慧連結** 特性簡化在信件中插入長 URL 的過程。 應用程式自動縮短了您的 URL , 提供了一個重要的優勢 —— 能夠 **跟蹤和儲存移動編號** 當收件人與智慧連結互動時。

## 編譯智慧短訊息

![Compose Smart SMS](images/smartsms2.png)

### 建立 A2P 智慧短訊息運動
導航到 **智慧短訊息** 選項來訪問作曲頁面,您可以在其中剪接 **A2P (個人申請)** 智慧短訊息運動.

#### 運動名稱
- 讓你的競選活動有一個友好的名字。
- 以當前日期時間和 <span data-ph="0"></span> 字首。

#### 發件人標識
- 從下放選單中選擇一個經批准的發件人ID.
- 如果 **打開發件人 ID** 選項,在文字框中輸入動態發件人ID。
- 此ID 顯示為收件人手機上的傳送地址 。

#### 聯絡人
- 從組中選擇聯絡人,上傳本地檔案,或手動輸入聯絡人.
- 移動編號應用逗號分隔,從國家程式碼開始(不包含 <span data-ph="0"></span>) (中文(簡體) ).
- 可選擇啟用 **自動新增國家程式碼** 下劃線 *我的簡介* 選項卡。

#### 輸入信件
- 點選“ Enter Message” 文字框訪問最後5條訊息 。
- 底部的計數器表示信件長度和計數 。

#### 草稿和模板
- 從儲存的草稿或模板中選擇內容 。
- 草案可以完全編輯,而模板修改僅限於佔位符.

#### 閃光
- 啟用 **閃光** 選項,將信件直接傳送到手提電腦螢幕。

#### 統一碼
- 自動檢測Unicode訊息內容並檢查Unicode框.

#### 時間表
- 將您的訊息儲存為草稿或未來執行的時間表 。
- 您配置檔案的時區預設值 。

## 插入智慧連結

### 在您的短訊息中嵌入 URL
點選插入連結圖示到:
- 輸入網頁連結 URL **或者說**
- 上傳檔案

智慧短訊息引擎將 URL 縮短為 a **智慧連結**,無縫地嵌入到短訊息。

![Insert Link](images/smartsms3.png) 
![Link Example](images/smartsms4.png)

#### 其他選項 :
- **在組中新增聯絡人** ——單擊智慧連結時,自動將收件人的移動號碼新增到被選中的組中.
- **向代理傳送短訊息通知** ——當收件人點選智慧連結時,向被選中的代理傳送簡訊提示.

在插入連結後,信件內容中會出現被縮短的URL. 
點選 **傳送** 執行智慧短訊息運動。

## 管理代理和自動響應器

### 管理代理
這個 **管理代理** 特性允許在收件人點選智慧連結時向選定的代理傳送短訊息提醒。

![Manage Agents](images/smartsms10.png)

**步驟 :**
1. 輸入代理商的名稱和電子郵件地址 。
2. 點選 **儲存**。 。 。 。
3. 向輸入的電子郵件傳送檢察官辦公室。
4. 特工核查檢察官辦公室
5. 該代理人列於 **管理代理**。 。 。 。

![Agents List](images/smartsms11.png)

---

### 管理自動通訊員
這個 **管理自動通訊員** 選項允許您配置對代理商的自動回覆。

![Auto Responders](images/smartsms12.png)

#### 自動響應設定
- 新增一個 **標題** 給自動通訊員的
- 輸入 **發件人標識** 資訊傳送。
- 工藝 **訊息** 以傳送到選定的代理。

#### 動態內容的佔位符
使用佔位符用於 **手機號碼** 財務報告和審定財務報表 **日期時間** 向特工提供詳細資訊。

#### 工業應用
- 對旨在提高通訊效率的行業具有價值。
- 理想為 **鉛生成**。 。 。 。
- 啟用快速響應和自動化。

---

透過使用 **管理代理** 財務報告和審定財務報表 **自動響應器** 在Smart SMS中,各組織可以精簡溝通,確保及時回應,並最大限度地提高生產率.
