
# SMS intelligent

## Guide utilisateur de SMS intelligent

### Présentation
Bienvenue à **SMS intelligent**, la solution ultime pour la messagerie intelligente qui enrichit non seulement l'engagement mobile avec vos clients, mais vous permet également avec l'analyse de données de pointe. 
Ce guide complet vous guidera dans les fonctionnalités et fonctionnalités de Smart SMS, vous assurant de tirer le meilleur parti de cet outil puissant.

---

## Lien intelligent

### URL raccourcie avec suivi
Les **Lien intelligent** fonctionnalité simplifie le processus d'insertion de longues URLs dans vos messages. L'application raccourcit automatiquement l'URL pour vous, fournissant un avantage significatif — la capacité de **suivre et stocker les numéros mobiles** lorsque les destinataires interagissent avec le lien intelligent.

## Composer des SMS intelligents

![Compose Smart SMS](images/smartsms2.png)

### Création de campagnes SMS intelligentes A2P
Naviguez vers la **SMS intelligent** option d'accéder à la page de composition, où vous pouvez créer **A2P (Demande à personne)** campagnes SMS intelligentes.

#### Nom de la campagne
- Donnez à votre campagne un nom amical.
- Un nom par défaut est généré automatiquement avec la date actuelle et le <span data-ph="0"></span> Préfixe.

#### ID de l'expéditeur
- Choisissez un ID d'expéditeur approuvé dans le menu déroulant.
- Si la **Ouvrir l'identifiant de l'expéditeur** option est activée, entrez un identifiant d'expéditeur dynamique dans la zone de texte.
- Cet ID apparaît comme l'adresse de l'expéditeur sur le mobile du destinataire.

#### Personnes-ressources
- Sélectionnez des contacts de groupes, téléchargez des fichiers locaux ou entrez manuellement des contacts.
- Les numéros mobiles doivent être séparés par des virgules, en commençant par un code de pays (sans <span data-ph="0"></span>).
- En option, activer **Ajout automatique du code de pays** sous la *Mon profil* onglet.

#### Saisissez le message
- Accédez aux 5 derniers messages en cliquant sur la zone de texte "Entrer un message".
- Un compteur au bas indique la longueur du message et le nombre.

#### Projets et modèles
- Choisissez le contenu des ébauches ou des modèles enregistrés.
- Les ébauches peuvent être entièrement modifiées, tandis que les modifications du modèle sont limitées aux titulaires de place.

#### Flash
- Activer **Flash** option pour livrer des messages directement à l'écran du combiné.

#### Unicode
- Détectez automatiquement le contenu du message Unicode et cochez la case Unicode.

#### Tableau
- Sauvegardez votre message en tant que projet ou calendrier pour l'exécution future.
- Par défaut à votre profil dans le fuseau horaire.

## Insérer un lien intelligent

### Intégrer les URL dans votre SMS
Cliquez sur l'icône du lien vers :
- Saisissez une URL de lien web **ou**
- Télécharger un fichier

Le moteur SMS intelligent raccourcit l'URL en un **lien intelligent**, intégré parfaitement dans le SMS.

![Insert Link](images/smartsms3.png) 
![Link Example](images/smartsms4.png)

#### Options supplémentaires:
- **Ajouter des contacts dans les groupes** — Ajouter automatiquement le numéro mobile du destinataire à un groupe sélectionné lorsqu'il clique sur le lien intelligent.
- **Envoyer une notification SMS à Agent** — Envoyer une alerte SMS à un agent sélectionné lorsque le destinataire clique sur le lien intelligent.

Après avoir inséré le lien, l'URL raccourcie apparaît dans le contenu du message. 
Cliquez sur **Envoyer** pour exécuter votre campagne SMS Smart.

## Gérer les agents et les répondeurs automatiques

### Gérer les agents
Les **Gérer les agents** fonctionnalité permet d'envoyer des alertes SMS à des agents sélectionnés si les destinataires cliquent sur le lien intelligent.

![Manage Agents](images/smartsms10.png)

**Étapes :**
1. Entrez le nom et l'adresse électronique de l'agent.
2. Cliquez sur **Enregistrer**.
3. Un OTP est envoyé à l'email saisi.
4. L'agent vérifie le BTP.
5. L'agent est listé dans **Gérer les agents**.

![Agents List](images/smartsms11.png)

---

### Gérer les réponses automatiques
Les **Gérer les réponses automatiques** option vous permet de configurer les réponses automatisées aux agents.

![Auto Responders](images/smartsms12.png)

#### Configuration du répondeur automatique
- Ajouter un **titre** pour l'autorépondeur.
- Saisissez la **ID de l'expéditeur** pour la livraison de messages.
- Artisanat **message** à envoyer à l'agent sélectionné.

#### Placeholders for Dynamic Content
Utiliser des porte-places pour **numéro de téléphone mobile** et **date-heure** donner aux agents des informations détaillées.

#### Applications industrielles
- Valable pour les industries visant à améliorer l'efficacité de la communication.
- Idéal pour **production de plomb**.
- Active les réponses rapides et l'automatisation.

---

En utilisant **Gérer les agents** et **Répondeurs automatiques** dans le SGS intelligent, les organisations peuvent rationaliser la communication, assurer des réponses en temps opportun et maximiser la productivité.
