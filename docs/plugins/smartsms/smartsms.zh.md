## 编译智能短消息

![Compose Smart SMS](images/smartsms2.png)

### 创建 A2P 智能短消息运动
导航到 **智能短消息** 选项来访问作曲页面,您可以在其中剪接 **A2P (个人申请)** 智能短消息运动.

#### 运动名称
- 让你的竞选活动有一个友好的名字。
- 以当前日期时间和 <span data-ph="0"></span> 前缀。

#### 发件人标识
- 从下放菜单中选择一个经批准的发件人ID.
- 如果 **打开发件人 ID** 选项,在文本框中输入动态发件人ID。
- 此ID 显示为收件人手机上的发送地址 。

#### 联系人
- 从组中选择联系人,上传本地文件,或手动输入联系人.
- 移动编号应用逗号分隔,从国家代码开始(不包含 <span data-ph="0"></span>) (中文(简体) ).
- 可选择启用 **自动添加国家代码** 下划线 *我的简介* 选项卡。

#### 输入信件
- 点击“ Enter Message” 文本框访问最后5条消息 。
- 底部的计数器表示信件长度和计数 。

#### 草稿和模板
- 从保存的草稿或模板中选择内容 。
- 草案可以完全编辑,而模板修改仅限于占位符.

#### 闪光
- 启用 **闪光** 选项,将信件直接传送到手提电脑屏幕。

#### 统一码
- 自动检测Unicode消息内容并检查Unicode框.

#### 时间表
- 将您的消息保存为草稿或未来执行的时间表 。
- 您配置文件的时区默认值 。

## 插入智能链接

### 在您的短消息中嵌入 URL
点击插入链接图标到:
- 输入网页链接 URL **或者说**
- 上传文件

智能短消息引擎将 URL 缩短为 a **智能链接**,无缝地嵌入到短消息。

![Insert Link](images/smartsms3.png) 
![Link Example](images/smartsms4.png)

#### 其他选项 :
- **在组中添加联系人** ——单击智能链接时,自动将收件人的移动号码添加到被选中的组中.
- **向代理发送短消息通知** ——当收件人点击智能链接时,向被选中的代理发送短信提示.

在插入链接后,信件内容中会出现被缩短的URL. 
点击 **发送** 执行智能短消息运动。

## 管理代理和自动响应器

### 管理代理
这个 **管理代理** 特性允许在收件人点击智能链接时向选定的代理发送短消息提醒。

![Manage Agents](images/smartsms10.png)

**步骤 :**
1. 输入代理商的名称和电子邮件地址 。
2. 点击 **保存**。 。 。 。
3. 向输入的电子邮件发送检察官办公室。
4. 特工核查检察官办公室
5. 该代理人列于 **管理代理**。 。 。 。

![Agents List](images/smartsms11.png)

---

### 管理自动通讯员
这个 **管理自动通讯员** 选项允许您配置对代理商的自动回复。

![Auto Responders](images/smartsms12.png)

#### 自动响应设置
- 添加一个 **标题** 给自动通讯员的
- 输入 **发件人标识** 信息发送。
- 工艺 **消息** 以发送到选定的代理。

#### 动态内容的占位符
使用占位符用于 **手机号码** 财务报告和审定财务报表 **日期时间** 向特工提供详细信息。

#### 工业应用
- 对旨在提高通信效率的行业具有价值。
- 理想为 **铅生成**。 。 。 。
- 启用快速响应和自动化。

---

通过使用 **管理代理** 财务报告和审定财务报表 **自动响应器** 在Smart SMS中,各组织可以精简沟通,确保及时回应,并最大限度地提高生产率.
