
# Avis Événements

Les **"Notifications"** module dans iTextPro sert de système de suivi proactif, évaluant régulièrement les paramètres critiques de l'application sur différents points de données. 
Ce module est conçu pour déclencher des alertes en temps opportun aux intervenants concernés par courriel, en assurant une action rapide et en empêchant l'escalade des problèmes potentiels.
