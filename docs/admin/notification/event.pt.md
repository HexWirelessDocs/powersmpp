
# Notificações Eventos

A **"Notificações"** módulo em iTextPro serve como um sistema de monitoramento proativo, avaliando regularmente parâmetros críticos da aplicação em vários pontos de dados. 
Este módulo foi projetado para desencadear alertas oportunos para as partes interessadas relevantes via e-mail, garantindo ações rápidas e evitando que problemas potenciais aumentem.
