
# Notificaciones Eventos

El **"Notificaciones"** El módulo iTextPro sirve como sistema de monitoreo proactivo, evaluando regularmente los parámetros críticos de la aplicación en diversos puntos de datos. 
Este módulo está diseñado para activar alertas oportunas a las partes interesadas pertinentes mediante correo electrónico, garantizando la adopción de medidas rápidas y evitando la intensificación de los problemas potenciales.
