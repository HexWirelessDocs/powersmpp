
## Paramètres généraux

Dans **Paramètres généraux**, vous pouvez trouver quatre sous-options:

1. Gérer les paramètres généraux 
2. Forcer le changement de mot de passe 
3. Verrouillage forcé 

---

### Gérer les paramètres généraux

Les **Gérer les paramètres généraux** fonctionnalité dans iTextPRO permet aux administrateurs de configurer différents paramètres globaux pour améliorer la fonctionnalité et l'expérience utilisateur dans l'application.

---

---

#### Options de configuration clés :

- **Afficher les messages échoués dans l'affichage de la liste :** 
 Lorsqu'il est activé, iTextPRO affiche tous les messages échoués dans les comptes des utilisateurs sous le rapport d'affichage de la liste, montrant des détails tels que Numéro mobile, Message, Date de soumission, Date de fin, État du message et Code d'erreur.

- **Activer la protection contre les pertes sur la passerelle d'échec :** 
 Active la protection contre les pertes commerciales potentielles dues à des erreurs de configuration dans le prix de vente du compte utilisateur. Cette protection s'étend à la passerelle de secours.

- **Master Failover Gateway :** 
 Définit une passerelle globale de basculement. Si la passerelle principale fait face à une panne, le trafic SMS pour toute connectivité est acheminé vers la passerelle de sortie.

- **Avis d'alerte de faible crédit :** 
 Définit un seuil global pour les alertes électroniques à faible crédit. Les utilisateurs reçoivent des notifications lorsque leur solde de compte dépasse ce seuil, ce qui entraîne une recharge.

- **Limite de seuil de la file d'attente :** 
 Configure un seuil pour les messages dans la file d'attente de la passerelle. Une fois atteint, une alerte e-mail est envoyée à l'administrateur avec les détails de la passerelle et des messages en attente.

- **Nombre maximal d'alertes par jour :** 
 Limite le nombre d'alertes par courriel par jour pour prévenir le spam et gérer les notifications fréquentes pendant les pannes ou les événements similaires.

- **Intervalle minimal entre deux alertes :** 
 Définit l'intervalle de temps minimum (en minutes) entre les alertes email successives pour éviter les notifications excessives.

- **Pays et fuseau horaire par défaut :** 
 Définit les valeurs par défaut du pays et du fuseau horaire affichées lors de l'inscription de l'utilisateur et des options « Ajouter un nouvel utilisateur », ce qui permet aux utilisateurs de sélectionner leurs préférences et d'assurer un affichage exact de la date et de l'heure.

- **Ouvrez l'ID de l'expéditeur et les modèles pour les utilisateurs :** 
 Active ces options par défaut lors de la création de nouveaux utilisateurs.

- **Seuil global de survente :** 
 Limite de découvert préconfigurée dans le compte admin qui peut être appliqué aux comptes utilisateurs.

---

Ces paramètres contribuent collectivement à la personnalisation, à l'efficacité et au contrôle au sein d'iTextPRO, assurant une expérience personnalisée et transparente pour les administrateurs et les utilisateurs.
