
## Gerenciar grupos de usuários

Este recurso facilita o **agrupamento lógico de usuários**, permitindo o mapeamento desses grupos para específicos **MT regras de encaminhamento**.

Em caso de **Insuficiência** ou **problemas de desempenho** com o principal fornecedor de gateway, os usuários podem **mudar para um fornecedor de gateway alternativo** com um único clique.

![User Groups](images/usergroup1.png)

Ao actualizar o **MT roteamento diretamente do motor de roteamento principal**, os utilizadores podem rapidamente **transição do tráfego de mensagens para um fornecedor de gateway secundário**. Este processo simplificado garante **rápida resposta a quaisquer problemas**, reforço **fiabilidade do sistema** e **minimizando o tempo de inatividade**.
