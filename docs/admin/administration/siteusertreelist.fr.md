
## Liste des arbres utilisateurs du site

Les **fonctionnalité de hiérarchie des utilisateurs** fournit une **vue hiérarchique** des deux **utilisateurs et revendeurs**, permettant aux utilisateurs d'avoir une **vue d'ensemble de tous les clients** simultanément.

En sélectionnant un **Nom d'utilisateur dans la hiérarchie**, les utilisateurs peuvent sans effort **naviguer vers les informations de base du profil** du compte utilisateur correspondant. Ce processus simplifié permet **accès rapide à des informations détaillées** sur les utilisateurs individuels, **gestion des utilisateurs** et **navigation** dans le système.
