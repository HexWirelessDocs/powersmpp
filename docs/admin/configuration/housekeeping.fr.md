
## Ménage

Les **Ménage** fonctionnalité dans iTextPRO offre un outil puissant pour **réinitialiser les données de facturation principale** en une seule action. Cette option est destinée à une utilisation administrative dans les situations où une nouvelle configuration ou un nettoyage des données de facturation est nécessaire.

!!! danger "Important"
 Une fois supprimé, données **ne peut être restauré**. Utilisez cette fonctionnalité avec une extrême prudence.

---

### Comment utiliser le ménage

1. Naviguez vers la **Ménage** Chapitre.
2. Vous verrez une liste de **Tableaux de données de facturation principale** avec interrupteurs à bascule marqués **Oui/Non** à côté de chacun.
3. Pour réinitialiser les données d'une table spécifique, basculez le basculement vers **Oui**.
4. Après avoir sélectionné les tables nécessaires, cliquez sur **Réinitialiser** bouton.

---

### Avertisseur principal de données

En cliquant sur le bouton Réinitialiser :

- iTextPRO affiche une **Alerte de données principale** C'est parti.
- Vous serez invité avec une option de **prendre une sauvegarde** de vos données avant de procéder.
- Après confirmation, iTextPRO **supprimer définitivement** les données de facturation sélectionnées.

!!! warning "Sauvegarde recommandée"
 C'est **fortement recommandé** pour sauvegarder vos données avant d'utiliser cette fonctionnalité.

---

### Quand utiliser le ménage

- Lors de la mise en place d'une **environnement de facturation frais**.
- Pendant les **nettoyage des données** ou **Suppression des données d'essai**.
- Après exécution **migrations système** ou **réinitialise l'environnement**.

---

Le ménage est un **rarement utilisé mais critique** fonctionnalité dans iTextPRO pour la maintenance du système et l'hygiène des données. Toujours procéder avec soin et s'assurer que vous avez des sauvegardes appropriées en place avant d'exécuter une réinitialisation.
