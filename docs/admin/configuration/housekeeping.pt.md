
## Serviço de limpeza

A **Serviço de limpeza** recurso no iTextPRO oferece uma ferramenta poderosa para **reset Master Billing Data** em uma única ação. Esta opção destina-se a uso administrativo em situações em que é necessária uma nova configuração ou limpeza dos dados de faturamento.

!!! danger "Importante"
 Uma vez apagados, dados **não pode ser restaurado**. Use este recurso com extrema cautela.

---

### Como usar a limpeza

1. Navegar para o **Serviço de limpeza** secção.
2. Você verá uma lista de **Tabelas de dados de faturamento mestre** com interruptores marcados **Sim/Não** ao lado de cada um.
3. Para repor os dados de uma tabela específica, mude para **Sim.**.
4. Depois de selecionar as tabelas necessárias, clique no **Reiniciar** Botão.

---

### Alerta de dados mestre

Ao clicar no botão Reiniciar:

- iTextPRO exibe a **Alerta de dados mestre** Aparece.
- Você será solicitado com uma opção para **fazer um backup** de seus dados antes de prosseguir.
- Após confirmação, o iTextPRO **apagar permanentemente** os dados de faturamento selecionados.

!!! warning "Backup Recomendado"
 Está. **fortemente recomendado** para fazer backup de seus dados antes de usar este recurso.

---

### Quando usar a limpeza

- Ao configurar um **ambiente de faturamento fresco**.
- Durante grandes **limpeza dos dados** ou **remoção de dados de teste**.
- Após executar **migrações do sistema** ou **ambiente reinicia**.

---

A limpeza é uma **raramente utilizado mas crítico** característica no iTextPRO para manutenção do sistema e higiene de dados. Sempre proceda com cuidado e certifique-se de que você tem backups apropriados no local antes de executar uma reinicialização.
