
## 內部管理

這個 **內部管理** iTextPRO 中的特性提供了一個強大的工具 **重置主計費資料** 在一次行動中。 在需要重新設定或清理計費資料的情況下,這一備選辦法將供行政部門使用。

!!! danger "重要"
 一旦刪除,資料 **無法恢復**。非常謹慎地使用此特性。

---

### 如何使用家庭管理

1. 導航到 **內部管理** 節。
2. 你會看到一個列表 **主計費資料表** 帶有標籤的切換開關 **有/無** 每一個旁邊。
3. 要重置特定表格的資料, 請切換到 **對**。 。 。 。
4. 在選擇必要的表格後,單擊 **重設** 按鈕。

---

### QQ 主資料提醒

點選重置按鈕時 :

- iTextPRO 顯示一個 **主資料提醒** 彈出.
- 您將被提示一個選項: **帶備份** 在開始前你的資料
- 在確認後, iTextPRO 將 **永久刪除** 選定的計費資料。

!!! warning "建議備份"
 沒錯 **強烈建議** 在使用此特性之前備份您的資料。

---

### 何時使用管家

- 當設定一個 **新的收費環境**。 。 。 。
- 大專期間 **資料清理** 或者說 **測試資料去除**。 。 。 。
- 表演結束後 **系統遷移** 或者說 **環境重置**。 。 。 。

---

管家是個 **很少使用但很關鍵** 用於系統維護和資料衛生的 iTextPRO 特性。 在執行重置前, 始終小心行事, 並確保您有適當的備份 。
