# 標準 SMPP 錯誤程式碼列表

本頁面提供了所有SMPP協議出錯程式碼,iTextPro應用出錯程式碼和iTextHub HTTP Gateway出錯程式碼的完整參考. 使用此引用來診斷報告,PDU日誌和事件檢視器中看到的訊息傳送失敗.

!!! tip "如何使用這一參考"
    - 當你看到您的錯誤程式碼 **運動報告**, (中文(簡體) ). **PDU 日誌**,或 **事件檢視器**在上表中查詢
    - **標準 SMPP 程式碼** (0-255)由外部簡訊中心/出入口返回
    - **iTextPro 程式碼** (1024-1096)由Power SMPP平臺內部生成.
    - **HTTP GW 程式碼** (110-1095)涉及HTTP閘道器連線問題
    - 關於閘道器特定出錯程式碼配置,參見 [Gateway出錯程式碼].(errorcode.md)
