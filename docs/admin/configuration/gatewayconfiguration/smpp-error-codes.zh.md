!!! tip "如何使用这一参考"
    - 当你看到您的错误代码 **运动报告**, (中文(简体) ). **PDU 日志**,或 **事件查看器**在上表中查找
    - **标准 SMPP 代码** (0-255)由外部短信中心/出入口返回
    - **iTextPro 代码** (1024-1096)由Power SMPP平台内部生成.
    - **HTTP GW 代码** (110-1095)涉及HTTP网关连接问题
    - 关于网关特定出错代码配置,参见 [Gateway出错代码].(errorcode.md)
