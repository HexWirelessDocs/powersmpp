## 管理接口

这个 **管理接口** iTextPRO 中的段落允许管理员通过 **SMPP 连接器( ESME 账户)** 财务报告和审定财务报表 **HTTP 连接器**. 这些连接器通过与外部系统的无缝整合而增强通信能力。

---

### 1. SMPP连接器(ESME账户)

![SMPP Connector](images/manage1.png)

这个 **SMPP 连接器**页:1 **MEME (外部短信息实体) 账户**,促进合作伙伴与 **SMPP 服务器** 在 iTextPRO 内。

#### 如何添加新的 SMPP 连接器 :
1. 搜索特定用户账户 。
2. 点击 **“添加新”** 启动配置。
3. 填写所需细节:

**ESME 账户设置域 :**

| 外地 | 说明 |
|-------|-------------|
| **系统标识** | 连接到无害环境管理账户的用户名 |
| **密码** | ESME 账户的认证密码 |
| **白名单 IP** | 只允许从此 IP 连接 |
| **Tx 计数** | 传送器会议次数(Tx) |
| **Rx 计数** | 接收者(Rx)会议次数 |
| **TRx 计数** | 收发报机(TRx)会议次数 |

#### 高级设置 :
- **验证 IP**: 启用源IP地址的验证. 只有列入白名单的IP可以连接.
- **账户已激活**:启用后,ESME用户可以连接到SMPP服务器.
- **列入黑名单**: 如果ESME 用户违反 **ESME 黑名单规则**。 。 。 。

### 2. HTTP 连接器

![HTTP Connector Setup](images/manage2.png)

这个 **HTTP 连接器** 允许合作伙伴使用 iTextPRO 集成 **基于 HTTP 的 API**。 。 。 。

#### 启用 HTTP API 访问的步骤 :
1. 启动 **API 状态** 在想要的用户账户中。
2. 一旦启动, **开发者 API** 区域在用户界面中可见。
3. 从那里,用户可以:
   - 视图 **API 证书**
   - **下载 API 文档** (PDF格式) (中文(简体) ).
   - 开始使用 HTTP API 端点来提交信件,报告,以及更多 。

![HTTP API Dashboard](images/manage3.png)

### 内容提要

这个 **管理接口** 在iTextPRO中为两者提供统一和直观的设置 **SMPP 和 HTTP 连接器**,允许:
- 合作伙伴和转售商的安全和控制准入。
- API 文档和工具,便于整合.
- 会话级别控制和基于IP的访问管理.

通过利用这些连接器,iTextPRO用户可以在保持严格控制与灵活性的同时,扩大他们的通讯范围.
