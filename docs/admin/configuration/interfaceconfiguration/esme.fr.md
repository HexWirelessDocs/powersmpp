
## Liste des clients ESME

Les **Liste des clients ESME** section dans iTextPRO fournit une vue centralisée de tous les utilisateurs qui ont opté pour **Services de la SMPP** (entités externes de messagerie courte). Cette interface permet aux administrateurs de surveiller, de rechercher et de gérer facilement les clients ESME connectés.

---

![ESME Client List](images/esmeclient1.png)

---

### Principales caractéristiques :

- **Aperçu du client**: 
 Affiche une liste de tous les utilisateurs ayant des comptes ESME (SMPP) actifs.

- **Fonctionnalité de la recherche**: 
 Un intuitif **boîte de recherche** permet de filtrer les comptes ESME sur la base:
  - **Nom d'utilisateur**
  - **ID système**

Cette fonctionnalité assure un accès rapide aux détails spécifiques du compte ESME sans naviguer dans de longues listes.

---

### Cas d'utilisation :

Si vous souhaitez trouver l'état de connexion ou les détails d'un utilisateur ESME avec l'ID système <span data-ph="0"></span>, entrez simplement cela dans la barre de recherche, et le système va filtrer automatiquement les résultats.

---

Les **Liste des clients ESME** est un utilitaire utile pour maintenir la visibilité et un accès rapide à tous les clients connectés SMPP dans votre environnement iTextPRO.
