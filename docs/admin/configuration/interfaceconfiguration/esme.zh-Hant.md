
## ESME 客戶列表

這個 **ESME 客戶列表** iTextPRO 的段落提供了所有選擇的使用者的集中檢視 **SMPP服務** (外部短訊息實體). 這種介面使管理員能夠方便地監測、搜尋和管理相連的ESME客戶端。

---

![ESME Client List](images/esmeclient1.png)

---

### 關鍵特性 :

- **客戶概覽**數字 : 
 顯示所有有活動ESME(SMPP)賬戶的使用者列表。

- **搜尋功能**數字 : 
 直覺 **搜尋框** 允許基於下列內容過濾ESME賬戶:
  - **使用者名稱**
  - **系統標識**

這一功能確保快速獲取具體的ESME賬戶細節而無需透過冗長的清單。

---

### 使用大小寫 :

如果您想找到一個ESMEE使用者與系統ID的連線狀態或細節 <span data-ph="0"></span>輸入搜尋欄,系統會自動過濾結果。

---

這個 **ESME 客戶列表** 在您的 iTextPRO 環境中保持所有 SMPP 連線客戶端的能見度和快速訪問的有用工具 。
