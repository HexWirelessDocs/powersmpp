
## ESME 黑名單規則

這個 **ESME 黑名單規則** 在 iTextPRO 中,它是保護您的 SMPP 伺服器免遭無賴或配置錯誤的ESME 客戶端的主動保障. 這些客戶端可能由於重複或異常的命令請求而導致效能退化. 透過設定這個規則,管理員可以 **自動黑名單** 可疑客戶,防止進一步損害。

---

### 目的

如果一個無賴的ESME使用者由於執行不當或意外的高頻請求而造成威脅,則這項規則使iTextPRO能夠永久地拒絕該使用者的請求,維持 **伺服器穩定性和完整性**。 。 。 。

---

### 配置選項

![Blacklist Rule Setup](images/blacklist1.png) 
![Blacklist Rule Fields](images/blacklist2.png)

配置新的ESME黑名單規則:

1. **規則名稱**數字 : 
   - 輸入黑名單規則的描述性友好名稱(例如, <span data-ph="0"></span>) (中文(簡體) ).

2. **使用者**數字 : 
   - 選擇規則應監測的ESME使用者賬戶。

3. **ESME 系統標識**數字 : 
   - 選擇具體 **系統標識** (a) 使用無害環境管理的使用者。

4. **命令型別**數字 : 
   - 選擇要監視的命令型別 :
     - **詢問連結**
     - **捆綁**

5. **時間間隔**數字 : 
   - 設定間隔時間 :
     - 小時
     - 分鐘
     - 秒數

6. **頻率**數字 : 
   - 定義選中的命令型別在指定的時間間隔內可以發生的次數 。

7. 填寫細節後,單擊 **"救贖"** 以啟用規則。
