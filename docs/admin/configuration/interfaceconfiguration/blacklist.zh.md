
## ESME 黑名单规则

这个 **ESME 黑名单规则** 在 iTextPRO 中,它是保护您的 SMPP 服务器免遭无赖或配置错误的ESME 客户端的主动保障. 这些客户端可能由于重复或异常的命令请求而导致性能退化. 通过设置这个规则,管理员可以 **自动黑名单** 可疑客户,防止进一步损害。

---

### 目的

如果一个无赖的ESME用户由于执行不当或意外的高频请求而造成威胁,则这项规则使iTextPRO能够永久地拒绝该用户的请求,维持 **服务器稳定性和完整性**。 。 。 。

---

### 配置选项

![Blacklist Rule Setup](images/blacklist1.png) 
![Blacklist Rule Fields](images/blacklist2.png)

配置新的ESME黑名单规则:

1. **规则名称**数字 : 
   - 输入黑名单规则的描述性友好名称(例如, <span data-ph="0"></span>) (中文(简体) ).

2. **用户**数字 : 
   - 选择规则应监测的ESME用户账户。

3. **ESME 系统标识**数字 : 
   - 选择具体 **系统标识** (a) 使用无害环境管理的用户。

4. **命令类型**数字 : 
   - 选择要监视的命令类型 :
     - **询问链接**
     - **捆绑**

5. **时间间隔**数字 : 
   - 设置间隔时间 :
     - 小时
     - 分钟
     - 秒数

6. **频率**数字 : 
   - 定义选中的命令类型在指定的时间间隔内可以发生的次数 。

7. 填写细节后,单击 **"救赎"** 以激活规则。
