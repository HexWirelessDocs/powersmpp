## 管理介面

這個 **管理介面** iTextPRO 中的段落允許管理員透過 **SMPP 聯結器( ESME 賬戶)** 財務報告和審定財務報表 **HTTP 聯結器**. 這些聯結器透過與外部系統的無縫整合而增強通訊能力。

---

### 1. SMPP聯結器(ESME賬戶)

![SMPP Connector](images/manage1.png)

這個 **SMPP 聯結器**頁:1 **MEME (外部簡訊息實體) 賬戶**,促進合作伙伴與 **SMPP 伺服器** 在 iTextPRO 內。

#### 如何新增新的 SMPP 聯結器 :
1. 搜尋特定使用者賬戶 。
2. 點選 **“新增新”** 啟動配置。
3. 填寫所需細節:

**ESME 賬戶設定域 :**

| 外地 | 說明 |
|-------|-------------|
| **系統標識** | 連線到無害環境管理賬戶的使用者名稱 |
| **密碼** | ESME 賬戶的認證密碼 |
| **白名單 IP** | 只允許從此 IP 連線 |
| **Tx 計數** | 傳送器會議次數(Tx) |
| **Rx 計數** | 接收者(Rx)會議次數 |
| **TRx 計數** | 收發報機(TRx)會議次數 |

#### 高階設定 :
- **驗證 IP**: 啟用源IP地址的驗證. 只有列入白名單的IP可以連線.
- **賬戶已啟用**:啟用後,ESME使用者可以連線到SMPP伺服器.
- **列入黑名單**: 如果ESME 使用者違反 **ESME 黑名單規則**。 。 。 。

### 2. HTTP 聯結器

![HTTP Connector Setup](images/manage2.png)

這個 **HTTP 聯結器** 允許合作伙伴使用 iTextPRO 整合 **基於 HTTP 的 API**。 。 。 。

#### 啟用 HTTP API 訪問的步驟 :
1. 啟動 **API 狀態** 在想要的使用者賬戶中。
2. 一旦啟動, **開發者 API** 區域在使用者介面中可見。
3. 從那裡,使用者可以:
   - 檢視 **API 證書**
   - **下載 API 文件** (PDF格式) (中文(簡體) ).
   - 開始使用 HTTP API 端點來提交信件,報告,以及更多 。

![HTTP API Dashboard](images/manage3.png)

### 內容提要

這個 **管理介面** 在iTextPRO中為兩者提供統一和直觀的設定 **SMPP 和 HTTP 聯結器**,允許:
- 合作伙伴和轉售商的安全和控制准入。
- API 文件和工具,便於整合.
