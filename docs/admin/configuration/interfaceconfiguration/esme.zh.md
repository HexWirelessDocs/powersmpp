
## ESME 客户列表

这个 **ESME 客户列表** iTextPRO 的段落提供了所有选择的用户的集中视图 **SMPP服务** (外部短消息实体). 这种接口使管理员能够方便地监测、搜索和管理相连的ESME客户端。

---

![ESME Client List](images/esmeclient1.png)

---

### 关键特性 :

- **客户概览**数字 : 
 显示所有有活动ESME(SMPP)账户的用户列表。

- **搜索功能**数字 : 
 直觉 **搜索框** 允许基于下列内容过滤ESME账户:
  - **用户名**
  - **系统标识**

这一功能确保快速获取具体的ESME账户细节而无需通过冗长的清单。

---

### 使用大小写 :

如果您想找到一个ESMEE用户与系统ID的连接状态或细节 <span data-ph="0"></span>输入搜索栏,系统会自动过滤结果。
