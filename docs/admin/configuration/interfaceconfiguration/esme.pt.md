
## Lista de Clientes ESME

A **Lista de Clientes ESME** seção no iTextPRO fornece uma visão centralizada de todos os usuários que optaram por **Serviços SMPP** (Entidades externas de mensagens curtas). Esta interface permite que os administradores monitorem, pesquisem e gerenciem facilmente clientes ESME conectados.

---

![ESME Client List](images/esmeclient1.png)

---

### Características chave:

- **Visão Geral do Cliente**: 
 Mostra uma lista de todos os usuários com contas ESME (SMPP) ativas.

- **Funcionalidade da Pesquisa**: 
 Intuitivo **caixa de pesquisa** permite filtrar contas ESME com base em:
  - **Utilizador**
  - **ID do sistema**

Este recurso garante acesso rápido a detalhes específicos da conta ESME sem navegar por longas listas.

---

### Caso de uso:

Se você quiser encontrar o status da conexão ou detalhes de um usuário do ESME com o ID do Sistema <span data-ph="0"></span>, basta inserir isso na barra de pesquisa, e o sistema irá auto-filtrar os resultados.

---

A **Lista de Clientes ESME** é um utilitário útil para manter visibilidade e acesso rápido a todos os clientes conectados ao SMPP em seu ambiente iTextPRO.
