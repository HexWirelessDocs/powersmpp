# Préparation aux tests vivants

Faciliter **Essais en cours** sur notre serveur, vous êtes tenu de partager votre **SMPP lier les pouvoirs** pour **SMS MO (d'origine mobile)** et **MT (Mobile Terminé)** les connexions. Ces références sont généralement obtenues de votre :

- **Fournisseur de passerelle SMS** ou 
- **SMSC (Centre de service des messages courts)**

Cette entité joue un **rôle vital** dans le stockage, la transmission, la conversion et la livraison de messages SMS aux combinés mobiles — généralement représentés par un **société de télécommunications** ou une **agrégateur tiers**.

## Exemple de paramètres de connexion SMPP

Votre typique **Téléphone ou SMSC** vous fournira les paramètres essentiels suivants:

| Paramètres | Requis | Désignation des marchandises |
|-----------|:--------:|-------------|
| **Adresse IP SMSC** | :material-check: | Adresse IP du serveur SMSC |
| **Port SMSC** | :material-check: | Numéro de port pour la connexion SMPP |
| **ID système** | :material-check: | Nom d'utilisateur pour l'authentification SMPP |
| **Mot de passe** | :material-check: | Mot de passe pour l'authentification SMPP |
| **Type de système** | | identificateur optionnel pour le système ESME |
| **Mode d'assemblage** | :material-check: | Émetteur / récepteur / émetteur |
| **Nombre de sessions** | :material-check: | Nombre de sessions Tx / Rx / TxRx |
| **TON** | | Type de nombre (p. ex., international, alphanumérique) |
| **NPI** | | Indicateur du plan de numérotation (par exemple, E.164, RNIS) |
