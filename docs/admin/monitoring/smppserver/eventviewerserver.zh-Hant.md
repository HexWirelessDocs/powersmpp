
# 事件檢視器 (伺服器)

這個 **事件檢視器 (伺服器)** 在 iTextPRO 中允許 **實時監測** 應用事件 **ESME 使用者**。 。 。 。 
它會捕捉到 **錯誤** 財務報告和審定財務報表 **墜機事件** 可能影響客戶,展示 **綜合日誌** 幫助有效排除出故障。

---

## 關鍵特性

### **錯誤和崩潰事件監測**
- 抓住影響無害環境管理使用者的應用事件。
- 顯示詳細錯誤和崩潰日誌,以便進行深入分析。

### **實時監測**
- 為伺服器側應用事件提供實時能見度.
- 能夠迅速發現和應對潛在的問題。

### **解決問題支援**
- 以完整的日誌細節促進有效解決問題。
- 提供見解,以迅速診斷和解決問題。

---

## 補充說明
- **時區對齊** - 事件日誌顯示在管理員的時區,以便準確跟蹤和參考。
- **最佳做法** - 定期監測事件檢視器(伺服器),以加強 **系統可靠性** 改進 **客戶滿意度**。 。 。 。
