
# 事件查看器( DLR)

这个 **iTextPRO 事件查看器( DLR)** 提供了宝贵的见解 **交付报告更新时遇到的错误**。 。 。 。 
它会捕捉到 **综合活动详情**,能够高效地排除出故障,并确保DLR处理无缝.

---

## 关键特性
- **DLR 事件监测** – 跟踪系统中所有 DLR 更新事件.
- **详细错误抓取** - 记录准确错误信息以供分析。
- **解决问题支持** ——促进快速准确解决问题.
- **管理时区** – 所有日志都按照管理员的时区显示.
