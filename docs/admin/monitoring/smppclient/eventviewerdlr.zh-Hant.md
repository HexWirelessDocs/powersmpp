
# 事件檢視器( DLR)

這個 **iTextPRO 事件檢視器( DLR)** 提供了寶貴的見解 **交付報告更新時遇到的錯誤**。 。 。 。 
它會捕捉到 **綜合活動詳情**,能夠高效地排除出故障,並確保DLR處理無縫.

---

## 關鍵特性
- **DLR 事件監測** – 跟蹤系統中所有 DLR 更新事件.
- **詳細錯誤抓取** - 記錄準確錯誤資訊以供分析。
- **解決問題支援** ——促進快速準確解決問題.
- **管理時區** – 所有日誌都按照管理員的時區顯示.
