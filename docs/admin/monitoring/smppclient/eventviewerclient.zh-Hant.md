
# 事件檢視器( 客戶端)

這個 **iTextPRO 事件檢視器( 客戶端)** 顯示器和日誌 **系統事件**,向管理員詳細介紹 **錯誤和崩潰** 這可能會影響簡訊的流量。 
這個工具對於 **積極解決問題** 財務報告和審定財務報表 **高效排除出錯**,幫助維持系統穩定.

---

## 關鍵特性
- **事件監測** - 跟蹤所有相關的 iTextPRO 系統事件。
- **詳細錯誤日誌** – 顯示完整的錯誤和崩潰細節用於診斷.
- **主動解決問題** ——能及早發現並解決問題.
- **管理時區** – 所有日誌根據管理員的時區可見.

---

## 福利
- **主動解決問題** - 在問題升級之前找出並解決問題。
- **解決問題** – 詳細日誌可以更快地進行根源分析.
- **保持系統穩定** ——確保簡訊平臺效能不間斷.
