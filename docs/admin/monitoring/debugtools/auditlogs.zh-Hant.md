
# 審計日誌

這個 **審計日誌** iTextPRO 記錄中的特性 a **綜合審計線索** 雙方開展的各種活動 **管理員** 財務報告和審定財務報表 **使用者賬戶**。 。 。 。 
該系統透過跟蹤關鍵系統活動,例如: **密碼恢復**, (中文(簡體) ). **屬性編輯/更新/刪除**,以及 **發件人標識** 或者說 **模板請求**。 。 。 。

---

## 金鑰細節已抓取

- **使用者名稱** - 確定執行事件的賬戶。 
- **執行日期** - 記錄事件的確切日期和時間。 
- **IP 地址** – 記錄動作的發源地IP地址. 
- **事件型別** – 分類事件的性質(例如密碼恢復,財產更新).
