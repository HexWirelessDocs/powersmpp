# Preparação para testes ao vivo

Facilitar **Ensaios ao vivo** em nosso servidor, você é obrigado a compartilhar seu **SMPP vincular credenciais** em vez **SMS MO (Mobile Originado)** e **MT (Terminado móvel)** ligações. Essas credenciais são tipicamente obtidas de seus:

- **Fornecedor de Gateway SMS** ou 
- **SMSC (Centro de Serviços de Mensagens Curtas)**

Esta entidade desempenha um **papel vital** no armazenamento, encaminhamento, conversão e entrega de mensagens SMS para aparelhos móveis — tipicamente representados por **empresa de telecomunicações** ou a **Agregador de terceiros**.

## Parâmetros de Ligação SMPP da Amostra

O teu típico **Telco ou SMSC** irá fornecer-lhe os seguintes parâmetros essenciais:

| Parâmetro | Requerido | Designação das mercadorias |
|-----------|:--------:|-------------|
| **Endereço IP do SMSC** | :material-check: | Endereço IP do servidor SMSC |
| **Porta SMSC** | :material-check: | Número da porta para conexão SMPP |
| **ID do sistema** | :material-check: | Utilizador da autenticação SMPP |
| **Senha** | :material-check: | Senha para autenticação SMPP |
| **Tipo de sistema** | | Identificador opcional para o sistema ESME |
| **Modo de ligação** | :material-check: | Transmissor / receptor / transmissor |
| **Número de Sessões** | :material-check: | Contagem de sessões Tx / Rx / TxRx |
| **TON** | | Tipo de número (por exemplo, internacional, alfanumérico) |
| **NPI** | | Indicador do plano de numeração (por exemplo, E.164, RDIS) |
