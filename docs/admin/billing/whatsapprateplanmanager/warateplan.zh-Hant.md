# WhatsApp 比率計劃配置

**導航 :** <span data-ph="0"></span> ➔ <span data-ph="1"></span> ➔ <span data-ph="2"></span> ➔ <span data-ph="3"></span>。 。 。 。

## 概覽

這個 **WhatsApp 比率計劃配置** 模組允許管理員定義 **定價模板** 對於什麼App訊息。 這些費率計劃指導如何在PowerSMPP平臺內對資訊進行計費和跟蹤,並分配給使用者賬戶進行準確的成本和收入管理。

---

## 建立新的“什麼”比率計劃

速率計劃制定過程的結構 **三個步驟**。 。 。 。

![Manage WA Rate Plans — List of configured plans](images/warateplan-01-list.png)

### 步驟1:基本細節

| 外地 | 說明 |
|-------|-------------|
| **速率計劃名稱** | 為計劃指定一個獨特的,描述性的名稱. |
| **活動** | 把計劃切換到 <span data-ph="0"></span> 或者說 <span data-ph="1"></span> 狀態。 |

![Add New WA Rate Plan — Step 1: Basic Details](images/warateplan-02-step1-basic.png)

### 第2步:價格細節(每個國家)

確定每個目的地國的定價結構。

![Add New WA Rate Plan — Step 2: Price Details](images/warateplan-03-step2-price.png)

| 外地 | 說明 |
|-------|-------------|
| **國家程式碼** | 選擇目的地國(例如, <span data-ph="0"></span>) (中文(簡體) ). 利率計劃支援多國定價。 |
| **價格** | 選定國家的WhatsApp資訊購買/批發成本。 |
| **銷售價格** | 每個WhatsApp訊息向終端使用者或客戶收取的零售價. |
| **平臺充電** | 附加平臺收費在銷售價格之上適用(如果適用)。 |

### 步驟3:指定使用者

一旦節省了費率計劃,就可以分配給個人使用者賬戶。 分配計劃可確保使用者傳送的所有WhatsApp訊息都按照配置的定價結構計費.

## WhatsApp 比率計劃配置的目的

!!! info "使用此模組..."
    - 定義針對特定國家的WhatsApp訊息定價.
    - 在一個單一模板內支援多國費率計劃。
    - 向特定使用者分配計費計劃。
    - 跟蹤WhatsApp訊息費用和每個使用者的收入.
    - 啟用WhatsApp流量的準確財務報告。
    - 管理平臺收費與成本和銷售價格。

!!! tip
 克隆是轉出現有計劃的一個區域變體——克隆母體,調整一到兩個國家的價格,並分配給新使用者組的最快方式.
