# WhatsApp 比率计划配置

**导航 :** <span data-ph="0"></span> ➔ <span data-ph="1"></span> ➔ <span data-ph="2"></span> ➔ <span data-ph="3"></span>。 。 。 。

## 概览

这个 **WhatsApp 比率计划配置** 模块允许管理员定义 **定价模板** 对于什么App消息。 这些费率计划指导如何在PowerSMPP平台内对信息进行计费和跟踪,并分配给用户账户进行准确的成本和收入管理。

---

## 创建新的“什么”比率计划

速率计划制定过程的结构 **三个步骤**。 。 。 。

![Manage WA Rate Plans — List of configured plans](images/warateplan-01-list.png)

### 步骤1:基本细节

| 外地 | 说明 |
|-------|-------------|
| **速率计划名称** | 为计划指定一个独特的,描述性的名称. |
| **活动** | 把计划切换到 <span data-ph="0"></span> 或者说 <span data-ph="1"></span> 状态。 |

![Add New WA Rate Plan — Step 1: Basic Details](images/warateplan-02-step1-basic.png)

### 第2步:价格细节(每个国家)

确定每个目的地国的定价结构。

![Add New WA Rate Plan — Step 2: Price Details](images/warateplan-03-step2-price.png)

| 外地 | 说明 |
|-------|-------------|
| **国家代码** | 选择目的地国(例如, <span data-ph="0"></span>) (中文(简体) ). 利率计划支持多国定价。 |
| **价格** | 选定国家的WhatsApp信息购买/批发成本。 |
| **销售价格** | 每个WhatsApp消息向最终用户或客户收取的零售价. |
| **平台充电** | 附加平台收费在销售价格之上适用(如果适用)。 |

### 步骤3:指定用户

一旦节省了费率计划,就可以分配给个人用户账户。 分配计划可确保用户发送的所有WhatsApp消息都按照配置的定价结构计费.

## WhatsApp 比率计划配置的目的

!!! info "使用此模块..."
    - 定义针对特定国家的WhatsApp消息定价.
    - 在一个单一模板内支持多国费率计划。
    - 向特定用户分配计费计划。
    - 跟踪WhatsApp消息费用和每个用户的收入.
    - 启用WhatsApp流量的准确财务报告。
    - 管理平台收费与成本和销售价格。

!!! tip
 克隆是转出现有计划的一个区域变体——克隆母体,调整一到两个国家的价格,并分配给新用户组的最快方式.
