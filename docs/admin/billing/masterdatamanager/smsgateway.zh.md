
# 管理网关价格

**优化iTextPRO中网关价格管理的财务准确性**

![Manage Gateway Price Overview](images/managegateway1.png)

在iTextPRO中,精确的财务跟踪和预防损失对于维持强有力的短信服务至关重要。 管理网关价格确保了准确的计费、成本跟踪和长期盈利。

---

## 网关成本价格的重要性

- **财务准确性** - 为已配置的网络运营商确定成本价格对准确的财务报告至关重要。 在对特定国家适用统一价格时,这一点尤其重要。
- **损失保护** - iTextPRO通过将网关成本价格以基准货币存储,有助于防止收入流失并防范价格差异。

---

## 配置网关成本价格

![Configure Gateway Cost Price](images/managegateway2.png)

### 1. 添加新的成本价格
- 导航到 **添加新成本价格** 在 iTextPRO 中。
- 配置细节 :
  - 网关供应商
  - 国家/管理协委会 -- -- MNC(使用) <span data-ph="0"></span> 固定成本价格)
  - 开始和结束日期
  - 覆盖现有定价的选项

---

### 2. 散装上传

![Bulk Upload](images/managegateway3.png)

Bulk上传功能一并简化了多个网关价格的设置.

#### 大块上传步骤

1. **准备 Excel 文件** 
   - 从 iTextPRO 下载样本 Excel 模板。 
   - 将所需的列填入您的网关价格列表。

 ![Prepare Excel File](images/managegateway4.png)

2. **选择网关** 
   - 选择要导入成本价格的网关 。 
   - 该系统显示任何现有成本价格以供参考。

3. **上传 Excel 文件** 
   - 点击 **选择文件** 上传您的 Excel 文件。 
   - 将列映射到:
     - **国家 ISO**
     - **价格**
     - **MCC - MNC 移动控制中心**

4. **提交** 
   - 点击 **提交** 以导入数据。

5. **查看导入价格** 
   - 选择网关并单击 **视图**。 。 。 。 
   - 审查进口价格的准确性。

![View Imported Prices](images/managegateway5.png)
