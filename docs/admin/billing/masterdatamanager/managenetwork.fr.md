
# Gérer les réseaux (MCC-MNC)

Les **Gérer le MCC-MNC** fonctionnalité dans iTextPRO est un outil clé pour configurer et gérer les opérateurs de réseau au sein du système. Il offre une interface intuitive et détaillée, rendant la gestion de l'opérateur simple et efficace.

![Manage Networks Overview](images/managenetwork1.png)

---

## Ajout d'un seul opérateur de réseau

![Add Network Operator](images/managenetwork2.png)

1. **Sélectionner un pays** – Choisissez le pays spécifique pour lequel vous souhaitez configurer un opérateur de réseau, en assurant la pertinence pour ce pays.
2. **Nom du réseau** – Entrez le nom de l'opérateur de réseau pour une identification claire.
3. **Configuration MCC-MNC** – 
   - **MCC:** Code pays mobile 
   - **MNC:** Code réseau mobile 
   - **Carte sauvage :** Utilisation <span data-ph="0"></span> pour un réseau plat sans préfixe.
4. **Ajout du processus** – Cliquez **Ajouter** pour sauver l'opérateur du réseau. Ce processus rationalise la configuration de l'opérateur et réduit la complexité.

---

## Fonctionnalité du chargement en vrac

![Bulk Upload](images/managenetwork3.png)

1. **Préparer le fichier Excel** – Téléchargez l'exemple de modèle Excel pour préparer votre liste d'opérateurs de réseau, en assurant le formatage correct des colonnes.
2. **Télécharger le fichier** – Utilisez la **Choisir un fichier** option pour télécharger le fichier Excel préparé et ajouter plusieurs opérateurs à la fois.
3. **Cartographie des colonnes** – Correspondez les colonnes Excel aux champs obligatoires tels que :
   - <span data-ph="0"></span>
   - <span data-ph="0"></span>
   - <span data-ph="0"></span>
4. **Soumettre et afficher** – Cliquez **Soumettre** pour traiter le téléchargement en vrac. Vous pouvez également personnaliser les enregistrements affichés dans la liste.

---

## Gestion des opérateurs existants

Les **Décision** menu vous permet :
- **Modifier** coordonnées de l'opérateur
- **Mise à jour** configurations
- **Supprimer** opérateurs

Cela vous assure de garder les paramètres de l'opérateur réseau précis et à jour en tout temps.
