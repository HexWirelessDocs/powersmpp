
# Gerenciar o Preço do Gateway

**Otimização da precisão financeira com a Gestão de Preços Gateway no iTextPRO**

![Manage Gateway Price Overview](images/managegateway1.png)

No iTextPRO, o rastreamento financeiro preciso e a prevenção de perdas são vitais para manter um serviço SMS robusto. Gerenciar preços de gateway garante faturamento preciso, monitoramento de custos e rentabilidade a longo prazo.

---

## Importância do preço do custo da porta

- **Precisão financeira** – Definir o preço de custo para operadores de rede configurados é essencial para relatórios financeiros precisos. Isto é especialmente importante quando se aplica um preço fixo a um determinado país.
- **Proteção contra perdas** – Ao armazenar o preço do custo do gateway na moeda base, o iTextPRO ajuda a evitar vazamentos de receita e salvaguardas contra discrepâncias de preços.

---

## Configurar Gateway Preço de Custo

![Configure Gateway Cost Price](images/managegateway2.png)

### 1. Adicionar novo preço de custo
- Navegar para **Adicionar novo preço de custo** em iTextPRO.
- Configurar detalhes:
  - Vendedor de gateway
  - País / MCC- MNC (usar <span data-ph="0"></span> por preço de custo fixo)
  - Datas de início e fim
  - Opção para substituir os preços existentes

---

### 2. Envio em massa

![Bulk Upload](images/managegateway3.png)

O recurso Bulk Upload simplifica a configuração de vários preços de gateway ao mesmo tempo.

#### Passos de envio em massa

1. **Preparar arquivo Excel** 
   - Baixe o modelo de exemplo Excel do iTextPRO. 
   - Preencha sua lista de preços Gateway com as colunas necessárias.

 ![Prepare Excel File](images/managegateway4.png)

2. **Selecionar Gateway** 
   - Escolha o gateway para o qual deseja importar preços de custo. 
   - O sistema exibe quaisquer preços de custo existentes para referência.

3. **Enviar o Ficheiro Excel** 
   - Clique **Escolher arquivo** para carregar seu arquivo Excel. 
   - Mapear as colunas para:
     - **País ISO**
     - **Preço de Custo**
     - **MCC-MNC**

4. **Enviar** 
   - Clique **Enviar** Importar os dados.

5. **Visualizando os preços importados** 
   - Selecione o gateway e clique **Ver**. 
   - Reveja os preços importados para obter precisão.

![View Imported Prices](images/managegateway5.png)

---

## Característica de Ação

A **Acção** recurso dá controle total sobre a gestão de preços gateway. 
Você pode:
- Editar os preços existentes 
- Atualizar configurações 
- Apagar os preços ultrapassados
