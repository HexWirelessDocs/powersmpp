
# 管理字首

這個 **管理字首** iTextPRO中的特性旨在高效處理與特定國家和網路運營商相關的移動數字字首。 它在確保準確的網路繪圖和計費方面發揮重要作用。

---

## 使用大小寫管理字首

### 計費結構考慮

#### 網路計費(MCC-MNC)
- **字首為對映引數** - 當計費基於網路(MCC-MNC)時,增加字首至關重要. 字首起到對映引數的作用,精確地識別出網路式計費模式.
- **強化計費精度** –字首管理確保計費系統與MCC-MNC程式碼所繫結的特定網路配置相配合.

#### 國別賬單
- **不需要字首管理** ——對於平地,以鄉為單位的收費,不需要字首. 賬單直接根據國家程式碼處理。
- **簡化計費程式** ——系統賬單交易只使用國家程式碼,精簡流程.

## 新增新字首

![Add Prefix](images/manageprefix2.png)

1. **選擇國家和運算子** - 選擇想要的國家和相關網路運營商。
2. **字首條目** – 貼上或手動將字首(沒有國家程式碼)輸入入輸入框.
3. **附加或替換選項** – 說吧 – 
   - 選擇 **對** 以新增現有列表中的新字首。 
   - 選擇 **沒有** 要完全替換現有的字首列表。
4. **提交資料** - 點選 **提交** 以最後確定字首匯入。

## 校驗字首功能

![Verify Prefix](images/manageprefix3.png)

這個 **校驗字首** 工具為給定的移動號碼提供快速網路細節。

1. **輸入移動編號** 輸入您想要驗證的號碼 。
2. **結果顯示** - 系統顯示:
   - 國家名稱
   - 網路名稱
   - MCC - MNC 移動控制中心

這有助於確認移動數字細節和相關網路資訊的準確性.
