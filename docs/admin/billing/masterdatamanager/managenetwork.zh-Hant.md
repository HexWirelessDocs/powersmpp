
# 管理網路(MCC-MNC)

這個 **管理 MCC- MNC** iTextPRO中的特性是系統內部配置和管理網路操作員的關鍵工具。 它提供了一個直觀而詳細的介面,使操作員管理簡單而高效.

![Manage Networks Overview](images/managenetwork1.png)

---

## 新增單個網路操作員

![Add Network Operator](images/managenetwork2.png)

1. **選擇國家** – 選擇您想要配置網路操作員的特定國家, 確保與該國的相關性 。
2. **網路名稱** – 輸入網路操作員的名稱以明確識別.
3. **MCC- MNC 配置** – 說吧 – 
   - **中導中心:** 移動國家程式碼 
   - **曼聯:** 行動網路程式碼 
   - **通關卡 :** 使用 <span data-ph="0"></span> 一個沒有字首的平面網路。
4. **新增程序** - 點選 **新增** 來儲存網路操作員。 這一過程精簡了操作員的設定並降低了複雜性.

---

## 大塊上傳功能

![Bulk Upload](images/managenetwork3.png)

1. **準備 Excel 檔案** – 下載樣本 Excel 模板以準備您的網路操作員列表,確保正確的列格式化.
2. **上傳檔案** - 使用 **選擇檔案** 選項,可以上傳準備的 Excel 檔案,並同時新增多個運算子。
3. **列對映** - 將 Excel 列與所需的欄位相匹配,例如:
   - <span data-ph="0"></span>
   - <span data-ph="0"></span>
   - <span data-ph="0"></span>
4. **提交和顯示** - 點選 **提交** 處理批次上傳。 您還可以自定義列表中顯示哪些記錄 。

---

## 管理現有操作員

這個 **行動** 選單允許您:
- **編輯** 運算子細節
- **更新** 配置
- **刪除** 運算子

這保證了您可以隨時保持網路操作員設定的準確性和最新性.
