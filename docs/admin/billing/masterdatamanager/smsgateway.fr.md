
# Gérer le prix de la passerelle

**Optimiser l'exactitude financière avec la gestion des prix dans iTextPRO**

![Manage Gateway Price Overview](images/managegateway1.png)

Dans iTextPRO, le suivi financier précis et la prévention des pertes sont essentiels pour maintenir un service SMS robuste. La gestion des prix des passerelles assure une facturation précise, un suivi des coûts et une rentabilité à long terme.

---

## Importance du prix de la passerelle

- **Précision financière** – La fixation du prix de revient pour les opérateurs de réseau configurés est essentielle pour une information financière précise. Ceci est particulièrement important lorsqu'on applique un prix forfaitaire pour un pays donné.
- **Protection contre les pertes** – En stockant le prix de la passerelle dans la devise de base, iTextPRO aide à prévenir les fuites de revenus et les garanties contre les écarts de prix.

---

## Configuration du prix de la passerelle

![Configure Gateway Cost Price](images/managegateway2.png)

### 1. Ajouter un nouveau prix
- Naviguez vers **Ajouter un nouveau prix** dans iTextPRO.
- Configurer les détails:
  - Fournisseur de passerelle
  - Pays / MCC-MNC (utilisation <span data-ph="0"></span> pour prix forfaitaire)
  - Dates de début et de fin
  - Option pour écraser les prix existants

---

### 2. Chargement en vrac

![Bulk Upload](images/managegateway3.png)

La fonction Bulk Upload simplifie la configuration de plusieurs prix de passerelle à la fois.

#### Étapes de chargement en vrac

1. **Préparer le fichier Excel** 
   - Télécharger l'exemple de modèle Excel depuis iTextPRO. 
   - Remplissez votre liste de prix Gateway avec les colonnes requises.

 ![Prepare Excel File](images/managegateway4.png)

2. **Sélectionner la passerelle** 
   - Choisissez la passerelle pour laquelle vous voulez importer les prix des coûts. 
   - Le système affiche les prix de référence existants.

3. **Télécharger le fichier Excel** 
   - Cliquez sur **Choisir un fichier** pour télécharger votre fichier Excel. 
   - Carter les colonnes à:
     - **Pays ISO**
     - **Prix**
     - **MCC-MNC**

4. **Soumettre** 
   - Cliquez sur **Soumettre** pour importer les données.

5. **Affichage des prix importés** 
   - Sélectionnez la passerelle et cliquez sur **Affichage**. 
   - Examiner les prix importés pour en vérifier l'exactitude.

![View Imported Prices](images/managegateway5.png)

## Fonction action

Les **Décision** fonctionnalité donne un contrôle complet sur la gestion des prix de passerelle. 
Vous pouvez :
- Modifier les prix existants 
- Mise à jour des configurations 
- Supprimer les prix périmés
