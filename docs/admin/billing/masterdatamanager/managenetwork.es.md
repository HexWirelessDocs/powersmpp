
# Gestionar redes (MCC-MNC)

El **Gestion MCC-MNC** característica en iTextPRO es una herramienta clave para configurar y gestionar operadores de red dentro del sistema. Ofrece una interfaz intuitiva y detallada, haciendo que la gestión del operador sea sencilla y eficiente.

![Manage Networks Overview](images/managenetwork1.png)

---

## Añadiendo un Operador de Red Única

![Add Network Operator](images/managenetwork2.png)

1. **Seleccionar país** – Elige el país específico para el que desea configurar un operador de red, asegurando la relevancia para ese país.
2. **Nombre de la red** – Introduzca el nombre del operador de red para una identificación clara.
3. **Configuración MCC-MNC** – 
   - **MCC:** Código del país móvil 
   - **MNC:** Código de red móvil 
   - **Wildcard:** Uso <span data-ph="0"></span> para una red plana sin prefijo.
4. **Proceso de adición** - Haga clic **Añadir** para salvar al operador de red. Este proceso simplifica la configuración del operador y reduce la complejidad.

---

## Funcionalidad de carga a granel

![Bulk Upload](images/managenetwork3.png)

1. **Prepare Excel File** – Descargue la plantilla de muestra Excel para preparar su lista de operadores de red, asegurando el formato de columna correcto.
2. **Cargar archivo** – Usar el **Elija el archivo** opción para subir el archivo Excel preparado y añadir varios operadores a la vez.
3. **Columna Mapping** – Columnas de Match Excel para campos obligatorios como:
   - <span data-ph="0"></span>
   - <span data-ph="0"></span>
   - <span data-ph="0"></span>
4. **Presentar y Mostrar** - Haga clic **Submit** para procesar la carga de vracs. También puede personalizar qué registros se muestran en la lista.

## Managing Existing Operators

El **Medida** menú te permite:
- **Editar** detalles del operador
- **Actualización** configuraciones
- **Suprimir** operadores

Esto asegura que puede mantener la configuración del operador de red precisa y actualizada en todo momento.
