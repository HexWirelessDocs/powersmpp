
# Gerenciar redes (MCC-MNC)

A **Gerenciar MCC-MNC** recurso no iTextPRO é uma ferramenta chave para configurar e gerenciar operadores de rede dentro do sistema. Ele oferece uma interface intuitiva e detalhada, tornando o gerenciamento do operador simples e eficiente.

![Manage Networks Overview](images/managenetwork1.png)

---

## Adicionar um Operador de Rede Único

![Add Network Operator](images/managenetwork2.png)

1. **Selecionar país** – Escolha o país específico para o qual deseja configurar um operador de rede, garantindo relevância para esse país.
2. **Nome da rede** – Digite o nome do operador de rede para uma identificação clara.
3. **Configuração do MCC-MNC** - Não. 
   - **MCC:** Código do país móvel 
   - **MNC:** Código da rede móvel 
   - **Wildcard:** Utilização <span data-ph="0"></span> para uma rede plana sem qualquer prefixo.
4. **Adicionando processo** – Clique **Adicionar** para salvar o operador de rede. Este processo simplifica a configuração do operador e reduz a complexidade.

---

## Funcionalidade de envio em massa

![Bulk Upload](images/managenetwork3.png)

1. **Preparar arquivo Excel** – Baixe o modelo de exemplo Excel para preparar sua lista de operadores de rede, garantindo a formatação correta da coluna.
2. **Enviar o Ficheiro** – Use o **Escolher arquivo** opção para carregar o arquivo Excel preparado e adicionar vários operadores de uma só vez.
3. **Mapeamento de Colunas** – Combine colunas do Excel com campos obrigatórios como:

4. **Enviar e Mostrar** – Clique **Enviar** para processar o upload em massa. Você também pode personalizar quais registros são exibidos na lista.

## Gestão de Operadores existentes

A **Acção** O menu permite- lhe:
- **Editar** detalhes do operador
- **Actualizar** configurações
- **Apagar** operadores

Isso garante que você pode manter as configurações do operador de rede precisas e atualizadas em todos os momentos.
