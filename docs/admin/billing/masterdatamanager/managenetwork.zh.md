
# 管理网络(MCC-MNC)

这个 **管理 MCC- MNC** iTextPRO中的特性是系统内部配置和管理网络操作员的关键工具。 它提供了一个直观而详细的界面,使操作员管理简单而高效.

![Manage Networks Overview](images/managenetwork1.png)

---

## 添加单个网络操作员

![Add Network Operator](images/managenetwork2.png)

1. **选择国家** – 选择您想要配置网络操作员的特定国家, 确保与该国的相关性 。
2. **网络名称** – 输入网络操作员的名称以明确识别.
3. **MCC- MNC 配置** – 说吧 – 
   - **中导中心:** 移动国家代码 
   - **曼联:** 移动网络代码 
   - **通关卡 :** 使用 <span data-ph="0"></span> 一个没有前缀的平面网络。
4. **添加进程** - 点击 **添加** 来保存网络操作员。 这一过程精简了操作员的设置并降低了复杂性.

---

## 大块上传功能

![Bulk Upload](images/managenetwork3.png)

1. **准备 Excel 文件** – 下载样本 Excel 模板以准备您的网络操作员列表,确保正确的列格式化.
2. **上传文件** - 使用 **选择文件** 选项,可以上传准备的 Excel 文件,并同时添加多个运算符。
3. **列映射** - 将 Excel 列与所需的字段相匹配,例如:
   - <span data-ph="0"></span>
   - <span data-ph="0"></span>
   - <span data-ph="0"></span>
4. **提交和显示** - 点击 **提交** 处理批量上传。 您还可以自定义列表中显示哪些记录 。

---

## 管理现有操作员

这个 **行动** 菜单允许您:
- **编辑** 运算符细节
- **更新** 配置
- **删除** 运算符

这保证了您可以随时保持网络操作员设置的准确性和最新性.
