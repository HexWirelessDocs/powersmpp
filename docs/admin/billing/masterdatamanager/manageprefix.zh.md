
# 管理前缀

这个 **管理前缀** iTextPRO中的特性旨在高效处理与特定国家和网络运营商相关的移动数字前缀。 它在确保准确的网络绘图和计费方面发挥重要作用。

---

## 使用大小写管理前缀

### 计费结构考虑

#### 网络计费(MCC-MNC)
- **前缀为映射参数** - 当计费基于网络(MCC-MNC)时,增加前缀至关重要. 前缀起到映射参数的作用,精确地识别出网络式计费模式.
- **强化计费精度** –前缀管理确保计费系统与MCC-MNC代码所绑定的特定网络配置相配合.

#### 国别账单
- **不需要前缀管理** ——对于平地,以乡为单位的收费,不需要前缀. 账单直接根据国家代码处理。
- **简化计费程序** ——系统账单交易只使用国家代码,精简流程.

## 添加新前缀

![Add Prefix](images/manageprefix2.png)

1. **选择国家和运算符** - 选择想要的国家和相关网络运营商。
2. **前缀条目** – 粘贴或手动将前缀(没有国家代码)输入入输入框.
3. **附加或替换选项** – 说吧 – 
   - 选择 **对** 以添加现有列表中的新前缀。 
   - 选择 **没有** 要完全替换现有的前缀列表。
4. **提交资料** - 点击 **提交** 以最后确定前缀导入。

## 校验前缀功能

![Verify Prefix](images/manageprefix3.png)

这个 **校验前缀** 工具为给定的移动号码提供快速网络细节。

1. **输入移动编号** 输入您想要验证的号码 。
2. **结果显示** - 系统显示:
   - 国家名称
   - 网络名称
   - MCC - MNC 移动控制中心

这有助于确认移动数字细节和相关网络信息的准确性.
