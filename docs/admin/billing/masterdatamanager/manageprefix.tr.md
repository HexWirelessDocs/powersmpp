
# Yönetin Prefix

The The The The The The The The **Yönetin Prefix** iTextPRO'de özellik, belirli ülkeler ve ağ operatörleri ile ilişkili mobil sayı eklerini verimli bir şekilde işlemek için tasarlanmıştır. Doğru ağ haritalama ve faturalama sağlamak için önemli bir rol oynar.

## Prefixes Yönetimi için Vaka Kullanımı

### Billing Structure Thinkations

#### Network-Based Billing (MCC-MNC)
- **Mapping parametresi olarak ek** - Fatura ağa (MCC-MNC) dayanıyorsa, eklerin eklenmesi önemlidir. Prefixes, ağ tabanlı faturalama kalıpları doğru bir şekilde tanımlamak için haritalama parametreleri olarak hareket eder.
- **Geliştirilmiş Billing Precision** - Ek yönetim, MCC-MNC kodlarına bağlı belirli ağ konfigürasyonları ile fatura sistemi hizalarını sağlar.

#### Ülke bazlı Billing
- **Tamamlama Yönetimi Gerekli** - Düz, ülke bazlı faturalama için, ekler gerekli değildir. Billing doğrudan ülke kodlarına dayanıyor.
- **Basitleştirilmiş Billing Process** - Sistem faturaları sadece ülke kodu kullanarak işlem yapar, süreci yayınlayın.

## Yeni ekleri eklemek

![Add Prefix](images/manageprefix2.png)

1. **Ülke ve Operatör Seçme** - İstenen ülkeyi ve ilişkili ağ operatörünü seçin.
2. **Prefix Giriş** - Paste veya manuel olarak giriş kutusunun ön satırına girer.
3. **Append veya Değiştir Seçenekleri** – 
   - Select Select Select Select **Evet Evet Evet** Mevcut listeye yeni ön eki indirmek için. 
   - Select Select Select Select **Hayır hayır hayır** Mevcut ek listesini tamamen değiştirmek için.
4. **Submission** - Click - Click **Gönder** Ön aktarmayı tamamlamak için.
