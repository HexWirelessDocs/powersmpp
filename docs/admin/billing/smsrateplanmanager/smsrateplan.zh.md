
# 速率计划模板

在有竞争力的短信业中,有效管理费率对最大限度地增加收入和确保可持续利润幅度至关重要。 iTextPRO 软件 **比率计划管理器** 实现过程自动化,实现准确、可扩展和易于管理的价格配置。

![Rate Plan Overview](images/rateplan1.png)

---

## 1. 出售价格的重要性

- **收入最大化** - 基于国家和MCC-MNC的准确费率确保最佳收入,同时平衡服务质量和成本。 
- **损失保护** - 以基准货币确定销售价格,防止收入流失,并保持一贯的计费准确性。

![Selling Price Importance](images/rateplan2.png)

---

## 2. 增加销售价格

### a. 默认比率计划应用
- 新用户账户自动继承 **违约率计划**,确保快速入职和一致性。

### b. 添加速率计划模板
1. 点击 **添加速率计划模板**。 。 。 。
2. 输入 a **友好名称** 用于模板。
3. 选择导入方法 :
   - **一个接一个**
   - **批量上传**

![Add Rate Plan Template](images/rateplan3.png)

---

## 3. 一个组合

- **国家/管理协委会-多国** - 选择国家和网络操作员。 
  - 使用 <span data-ph="0"></span> 全国所有网络的平价销售。
- **网关成本价格** - 查看 **最低** 财务报告和审定财务报表 **上限** 成本价格 **管理网关价格**。 。 。 。
- **完整列表视图** – 访问所有网关及其为所选网络配置的成本价格.

![View Gateway Cost Prices](images/rateplan4.png)

- **校验确认( V)** - 审查费率计划摘要并点击 **提交** 以保存更改。

---

## 4. 查看配置的销售价格

![View Configured Prices](images/rateplan5.png) 
- 点击 **视图** 将所有销售价格输入模板。 
- 使用此视图 **综合费率计划分析**。 。 。 。

![Detailed Price View](images/rateplan6.png)

---

## 5. 散装上传

![Bulk Upload Process](images/rateplan7.png)

**步骤 :**
1. **准备 Excel 文件**
   - 下载样本文件 。
   - 组织销售价格清单,提供所需细节。

2. **上传文件**
   - 点击 **选择文件** 上传您准备的 Excel。

3. **地图列**
   - 指定列用于:
     - **国家 ISO**
     - **销售价格**
     - **MCC - MNC 移动控制中心**

4. **选择销售价格公式**
   - 选项 :
     - 来自 Excel 的直接值
     - 数值 + {% of Value}
     - 数值 + {x}
     - 固定价值

![Select Formula](images/rateplan8.png)

5. **完成上传**
   - 确认并应用特定区域和网络的销售价格。

---

## 6. 批量编辑-快速销售价格更新

![Bulk Edit](images/rateplan9.png)

**步骤 :**
1. 点击 **批量编辑** 在比率计划管理器中。
2. 输入 **新销售价格** 在弹出。
3. 应用更改到 **多个网络同时出现**。 。 。 。

![Bulk Edit Popup](images/rateplan10.png)
