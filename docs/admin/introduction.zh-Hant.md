# 準備現場測試

便利 **現場測試** 在伺服器上,您需要共享您的 **SMPP 繫結證書** 頁:1 **SMS MO( 移動原始碼)** 財務報告和審定財務報表 **MT( 機動性終止)** 連線。 這些全權證書一般取自:

- **短訊息閘道器供應商** 或者說 
- **短訊息服務中心( Short Message Service Centre)**

此實體播放 **作用重大** 將短訊息儲存、轉發、轉換和傳送到移動話筒中——通常以 **電信公司** 或一個 **第三方合計**。 。 。 。

## SMPP 連線引數樣本

你那典型的 **電話或簡訊** 將為你們提供下列基本引數:

| 引數 | 需求 | 說明 |
|-----------|:--------:|-------------|
| **SMSC IP 地址** | :material-check: | SMSC 伺服器的IP 地址 |
| **SMSC 埠** | :material-check: | SMPP 連線的埠號 |
| **系統標識** | :material-check: | SMPP 認證的使用者名稱 |
| **密碼** | :material-check: | SMPP 認證密碼 |
| **系統型別** | | 無害環境管理系統的可選識別符號 |
| **繫結模式** | :material-check: | 傳送器/接收器/收發器 |
| **屆會次數** | :material-check: | Tx / Rx / TxRx 會話計數 |
| **頓** | | 數字型別(例如,國際,字母數字) |
| **NPI 狀態** | | 編號計劃指標(例如E.164、ISDN) |
