
# WhatsApp enviado contagem

A **Relatório WhatsApp** a seção foi integrada no módulo de relatórios principal para fornecer aos administradores visibilidade centralizada nos canais de comunicação.

Com esse aprimoramento, o administrador não precisa mais navegar para separar seções para diferentes canais. Se o relatório está relacionado com SMS ou WhatsApp, todas as funcionalidades de relatórios são acessíveis a partir do mesmo módulo Relatório.

Esta estrutura unificada garante um melhor monitoramento, acesso mais fácil aos dados históricos e revisão operacional simplificada.

---

## WhatsApp enviado contagem

A **WhatsApp enviado contagem** O relatório fornece um resumo estatístico rápido da atividade de mensagens.

Este relatório permite ao administrador:

- Selecione um intervalo de datas específico
- Ver o número total de mensagens WhatsApp enviadas dentro desse período
- Filtrar dados para um usuário específico, se necessário

Ao definir o intervalo de datas desejado, o sistema obtém e exibe a contagem total enviada para essa duração. Se um determinado usuário estiver selecionado, o relatório refletirá apenas o volume de mensagem do usuário.

![WhatsApp Sent Count Report](images/whatsappsentcount1.png)

### Casos de Uso

Este relatório é útil para:

- Revisão do uso de alto nível
- Monitorização do tráfego diário ou mensal
- Verificação rápida do consumo
- Referência básica de facturação
