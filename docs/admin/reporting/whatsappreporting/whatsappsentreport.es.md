
# WhatsApp Sent Report

El **WhatsApp Sent Report** proporciona información detallada a nivel de mensajes para un análisis más profundo.

A diferencia del informe del Conde enviado, que muestra números agregados, este informe muestra registros individuales de mensajes.

## Detalles del registro

Cada registro incluye típicamente:

- **Número de móvil** – Número de destinatario
- **Nombre del conector** – Plantilla usada
- **Mensaje** – Contenido del mensaje enviado
- **Fecha de presentación** – Fecha y hora de presentación
- **Costo del mensaje** – Costo por mensaje
- **Situación** – Estado de entrega actual
- **Código de error** – Códigos de error (si son aplicables)

## Características clave

- Filtro por **Nombre del usuario** para ver los mensajes de una cuenta específica
- Seleccione una **rango de fecha** para el análisis específico
- Usar el **Búsqueda** caja para filtrar palabras clave rápidas
- Ver detalles de entrega y fracaso en el nivel de mensaje individual

---

El **WhatsApp Sent Report** sirve como una ruta de auditoría completa para las transacciones de mensajes de WhatsApp, permitiendo a los administradores seguir las entregas, resolver fallos y revisar los detalles de mensajes sin esfuerzo.
