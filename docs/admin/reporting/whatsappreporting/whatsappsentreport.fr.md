
# Rapport de WhatsApp

Les **Rapport de WhatsApp** fournit des renseignements détaillés au niveau du message pour une analyse plus approfondie.

À la différence du rapport du compte d'envoi, qui montre les chiffres agrégés, ce rapport affiche les enregistrements de messages individuels.

## Détails

Chaque enregistrement comprend généralement:

- **Numéro mobile** – Numéro du bénéficiaire
- **Nom du connecteur** – Modèle utilisé
- **Message** – Contenu du message envoyé
- **Date de soumission** – Date et heure de soumission
- **Coût du message** – Coût par message
- **État** – État actuel de livraison
- **Code d'erreur** – Codes d'erreur (le cas échéant)

## Caractéristiques principales

- Filtrer par **Nom d' utilisateur** afficher les messages pour un compte spécifique
- Sélectionner un **date** pour une analyse ciblée
- Utilisez la **Rechercher** boîte de filtrage rapide des mots clés
- Afficher les détails de livraison et d'échec au niveau du message individuel

---

Les **Rapport de WhatsApp** sert de piste d'audit complète pour les transactions de messages WhatsApp, permettant aux administrateurs de suivre les livraisons, les défaillances de dépannage et de revoir les détails du message sans effort.
