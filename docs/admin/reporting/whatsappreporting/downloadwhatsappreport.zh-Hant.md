
# 下載WhatsApp報告

這個 **下載WhatsApp報告** 特性使管理員能夠以檔案格式生成和下載詳細的WhatsApp活動報告。

這一功能有助於:

- 業績分析
- 內部報告
- 核對賬單
- 與客戶共享結構化資料

它還確保WhatsApp和SMS競選記錄分開儲存,以便更加清晰和組織。

## WhatsApp 報告型別

要生成可下載的報告,管理員必須使用 **請求報告** 選項。

在要求報告時,管理員/使用者可以從下列報告型別中選擇:

### 1. 運動報告

運動報告詳細介紹了從申請中發起的運動。

本報告載有下列資料:

- 移動號碼
- 聯結器細節
- 信件內容
- 信件編號
- 信件狀態

本報告通常用於追蹤交付情況和核對。

### 2. 按鈕報告

如果一項運動包括了互動按鈕(如Opt-Out或其他動作按鈕),"按鈕報告"會捕捉到收件人的互動.

本報告包括:

- 收件人號碼
- 互動時間
- 被收件人點選按鈕

它幫助管理員分析使用者參與和反應行為.

### 3. 答覆報告

如果收件人對WhatsApp訊息做出回覆,這些回覆可以使用回覆報告選項下載.

本報告收錄:

- 收件人號碼
- 回覆內容
- 答覆日期和時間

它對審查客戶反應和維持通訊記錄很有用。

## 下載WhatsApp報告的步驟

遵循以下步驟生成並下載所需報告:

**步驟1:** 點選 **請求報告**
**步驟2:** 選擇想要的 **報表型別**
**步驟3:** 輸入報表的友好名稱
**步驟4:** 選擇正在要求報表的使用者名稱
**步驟5:** 輸入獲取報表的日期範圍
**步驟6:** 選擇連線符
**步驟7:** 點選 **請求報告** 提交

![Request Report Dialog](images/downloadwhatsapp2.png)

### 請求後

接到請求後:

- 系統將在背景中生成檔案.
- 報告條目將在UI中可見.
- 等待處理以完成並重新整理需要的頁面 。
- 當狀態改變到 **下載**,檔案將可用。
- 點選 **下載** 將報告儲存到您的本地系統。

---

## 內容提要

“WhatsApp報告”模組在一個統一的報告框架內提供簡要的見解和詳細的資訊分析。

有了Sent Counter,Sent Report等選項,以及可下載的Campaign,Button,和Reform reports等選項,管理員可以有效監控效能,跟蹤互動,並在整個WhatsApp運動中保持結構化的通訊記錄.
