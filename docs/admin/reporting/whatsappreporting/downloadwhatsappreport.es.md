
# Descargar WhatsApp Report

El **Descargar WhatsApp Report** característica permite a los administradores generar y descargar informes detallados de la campaña WhatsApp en formato de archivo.

Esta funcionalidad ayuda en:

- Análisis del desempeño
- Presentación de informes internos
- Conciliación
- Compartir datos estructurados con clientes

También garantiza que los registros de campañas WhatsApp y SMS se mantengan por separado para una mejor claridad y organización.

## Tipo de informe de WhatsApp

Para generar un informe descargable, el administrador debe utilizar el **Solicitud de informe** Opción.

Al solicitar un informe, el administrador/usuario puede seleccionar los siguientes tipos de informe:

### 1. Informe de campaña

El Informe de Campaña incluye detalles de las campañas iniciadas desde la aplicación.

El presente informe contiene información como:

- Número de móvil
- Detalles del conector
- Contenido del mensaje
- ID de mensaje
- Estado del mensaje

Este informe se utiliza típicamente para el seguimiento de la entrega y la reconciliación.

### 2. Button Report

Si una campaña incluye botones interactivos (como el Opt-Out u otros botones de acción), el Button Report captura las interacciones receptoras.

Este informe incluye:

- Número de destinatario
- Tiempo de interacción
- Botón clicado por el destinatario

Ayuda a los administradores a analizar el compromiso del usuario y el comportamiento de respuesta.

### 3. Informe de respuesta

Si los destinatarios responden a un mensaje WhatsApp, esas respuestas pueden descargarse utilizando la opción Responder Report.

Este informe recoge:

- Número de destinatario
- Responder contenido
- Fecha y hora de respuesta

Es útil para revisar las respuestas de los clientes y mantener los registros de comunicación.

---

## Pasos para descargar un informe de WhatsApp

Siga los pasos siguientes para generar y descargar el informe requerido:

**Paso 1:** Haga clic en **Solicitud de informe**
**Paso 2:** Seleccione el deseado **Tipo de informe**
**Paso 3:** Introduzca un nombre amistoso para el informe
**Paso 4:** Seleccione el nombre de usuario para el que se solicita el informe
**Paso 5:** Introduzca el rango de fechas para buscar el informe
**Paso 6:** Seleccione el conector
**Paso 7:** Haga clic en **Solicitud de informe** a presentar

![Request Report Dialog](images/downloadwhatsapp2.png)

### Después de solicitar

Una vez solicitado:

- El sistema generará el archivo en el fondo.
- La entrada del informe será visible en la UI.
- Espere a procesar para completar y refrescar la página si es necesario.
- Cuando el estado cambia a **Descargar**, el archivo estará disponible.
- Haga clic **Descargar** para guardar el informe a su sistema local.

---

## Resumen

El módulo de presentación de informes de WhatsApp proporciona información a nivel de resumen y análisis detallados a nivel de mensajes dentro de un marco unificado de presentación de informes.

Con opciones como Conde enviado, Informe enviado, y informes de Campaña descargable, Botón y Responder, los administradores pueden monitorear eficazmente el rendimiento, rastrear las interacciones y mantener registros de comunicación estructurados en las campañas de WhatsApp.
