
# 下载WhatsApp报告

这个 **下载WhatsApp报告** 特性使管理员能够以文件格式生成和下载详细的WhatsApp活动报告。

这一功能有助于:

- 业绩分析
- 内部报告
- 核对账单
- 与客户共享结构化数据

它还确保WhatsApp和SMS竞选记录分开保存,以便更加清晰和组织。

## WhatsApp 报告类型

要生成可下载的报告,管理员必须使用 **请求报告** 选项。

在要求报告时,管理员/用户可以从下列报告类型中选择:

### 1. 运动报告

运动报告详细介绍了从申请中发起的运动。

本报告载有下列资料:

- 移动号码
- 连接器细节
- 信件内容
- 信件编号
- 信件状态

本报告通常用于追踪交付情况和核对。

### 2. 按钮报告

如果一项运动包括了交互按钮(如Opt-Out或其他动作按钮),"按钮报告"会捕捉到收件人的交互.

本报告包括:

- 收件人号码
- 互动时间
- 被收件人点击按钮

它帮助管理员分析用户参与和反应行为.

### 3. 答复报告

如果收件人对WhatsApp消息做出回复,这些回复可以使用回复报告选项下载.

本报告收录:

- 收件人号码
- 回复内容
- 答复日期和时间

它对审查客户反应和维持通信记录很有用。

## 下载WhatsApp报告的步骤

遵循以下步骤生成并下载所需报告:

**步骤1:** 点击 **请求报告**
**步骤2:** 选择想要的 **报表类型**
**步骤3:** 输入报表的友好名称
**步骤4:** 选择正在要求报表的用户名
**步骤5:** 输入获取报表的日期范围
**步骤6:** 选择连接符
**步骤7:** 点击 **请求报告** 提交

![Request Report Dialog](images/downloadwhatsapp2.png)

### 请求后

接到请求后:

- 系统将在背景中生成文件.
- 报告条目将在UI中可见.
- 等待处理以完成并刷新需要的页面 。
- 当状态改变到 **下载**,文件将可用。
- 点击 **下载** 将报告保存到您的本地系统。

---

## 内容提要

“WhatsApp报告”模块在一个统一的报告框架内提供简要的见解和详细的信息分析。

有了Sent Counter,Sent Report等选项,以及可下载的Campaign,Button,和Reform reports等选项,管理员可以有效监控性能,跟踪互动,并在整个WhatsApp运动中保持结构化的通信记录.
