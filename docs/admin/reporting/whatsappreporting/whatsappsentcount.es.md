
# Cuento de WhatsApp enviado

El **WhatsApp Reporting** la sección se ha integrado en el módulo principal de presentación de informes para proporcionar a los administradores una visibilidad centralizada en todos los canales de comunicación.

Con esta mejora, el administrador ya no necesita navegar por secciones separadas para diferentes canales. Si el informe está relacionado con SMS o WhatsApp, todas las funcionalidades de presentación de informes son accesibles desde el mismo módulo de presentación de informes.

Esta estructura unificada garantiza una mejor vigilancia, un acceso más fácil a los datos históricos y un examen operacional simplificado.

---

## Cuento de WhatsApp enviado

El **Cuento de WhatsApp enviado** reporte proporciona un resumen estadístico rápido de la actividad de mensajería.

Este informe permite al administrador:

- Seleccione un rango de fecha específico
- Ver el número total de mensajes de WhatsApp enviados dentro de ese período
- Datos de filtro para un usuario específico, si es necesario

Al definir el rango de fecha deseado, el sistema sembra y muestra el recuento total enviado para esa duración. Si se selecciona un usuario en particular, el informe sólo reflejará el volumen del mensaje del usuario.

![WhatsApp Sent Count Report](images/whatsappsentcount1.png)

### Casos de uso

Este informe es útil para:

- Examen de la utilización de alto nivel
- Supervisión del tráfico diario o mensual
- Verificación del consumo rápido
- Referencia de facturación básica

---

El **Cuento de WhatsApp enviado** report ofrece una manera rápida y eficiente de monitorizar los volúmenes de mensajería WhatsApp a través de los usuarios y períodos de tiempo.
