
# 发送什么 App 数

这个 **WhatsApp 报告** 该科已纳入主要报告单元,使管理人员能够集中了解各通信渠道的情况。

通过这一增强,管理员不再需要为不同的频道导航以分离出分节. 无论报告与短信还是与WhatsApp有关,所有报告功能都可从同一报告模块中获取。

这一统一结构确保了更好的监测,便于获取历史数据,并简化了业务审查。

---

## 发送什么 App 数

这个 **发送什么 App 数** 报告提供了信息传递活动的快速统计摘要。

此报告允许管理员:

- 选择特定的日期范围
- 查看期间发送的WhatsApp信件总数
- 如果需要, 过滤特定用户的数据

通过定义理想的日期范围,系统获取并显示该期间发送的总数. 如果选择了特定的用户,报告将只反映该用户的信件卷.
