
# 傳送什麼 App 數

這個 **WhatsApp 報告** 該科已納入主要報告單元,使管理人員能夠集中瞭解各通訊渠道的情況。

透過這一增強,管理員不再需要為不同的頻道導航以分離出分節. 無論報告與簡訊還是與WhatsApp有關,所有報告功能都可從同一報告模組中獲取。

這一統一結構確保了更好的監測,便於獲取歷史資料,並簡化了業務審查。

---

## 傳送什麼 App 數

這個 **傳送什麼 App 數** 報告提供了資訊傳遞活動的快速統計摘要。

此報告允許管理員:

- 選擇特定的日期範圍
- 檢視期間傳送的WhatsApp信件總數
- 如果需要, 過濾特定使用者的資料

透過定義理想的日期範圍,系統獲取並顯示該期間傳送的總數. 如果選擇了特定的使用者,報告將只反映該使用者的信件卷.

![WhatsApp Sent Count Report](images/whatsappsentcount1.png)

### 使用大小寫

這份報告有助於:

- 高級別使用審查
- 監測每日或每月流量
- 快速消費核查
- 基本帳單參考
