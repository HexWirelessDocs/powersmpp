
# 傳送的WhatsApp 報告

這個 **傳送的WhatsApp 報告** 提供詳細的資訊,以便進行更深入的分析。

與顯示彙總數字的Sent Counter報告不同,本報告顯示的是單個信件記錄.

## 記錄細節

每個記錄通常包括:

- **移動號碼** - 收件人號碼
- **聯結器名稱** - 使用的模板
- **訊息** - 傳送信件內容
- **提交日期** - 提交日期和時間
- **信件成本** - 每封信的費用
- **狀態** - 目前交付情況
- **錯誤程式碼** - 錯誤程式碼(如果適用)

---

## 關鍵特性

- 過濾器由 **使用者名稱** 檢視特定賬戶的資訊
- 選擇一個 **日期範圍** 目標分析
- 使用 **搜尋** 用於快速關鍵字過濾的框
- 在單個信件級別檢視傳送和失敗細節

---

這個 **傳送的WhatsApp 報告** 作為WhatsApp訊息交易的全面審計線索,使管理員能夠追蹤訊息的交付、故障排除故障以及無心審查訊息細節。
