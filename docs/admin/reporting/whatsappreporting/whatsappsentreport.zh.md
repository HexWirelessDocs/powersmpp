
# 发送的WhatsApp 报告

这个 **发送的WhatsApp 报告** 提供详细的信息,以便进行更深入的分析。

与显示汇总数字的Sent Counter报告不同,本报告显示的是单个信件记录.

## 记录细节

每个记录通常包括:

- **移动号码** - 收件人号码
- **连接器名称** - 使用的模板
- **消息** - 发送信件内容
- **提交日期** - 提交日期和时间
- **信件成本** - 每封信的费用
- **状态** - 目前交付情况
- **错误代码** - 错误代码(如果适用)

---

## 关键特性

- 过滤器由 **用户名** 查看特定账户的信息
- 选择一个 **日期范围** 目标分析
- 使用 **搜索** 用于快速关键字过滤的框
- 在单个信件级别查看发送和失败细节

---

这个 **发送的WhatsApp 报告** 作为WhatsApp消息交易的全面审计线索,使管理员能够追踪消息的交付、故障排除故障以及无心审查消息细节。
