
# Baixar relatório do WhatsApp

A **Baixar relatório do WhatsApp** O recurso permite que os administradores gerem e baixem relatórios detalhados de campanha do WhatsApp em formato de arquivo.

Esta funcionalidade ajuda em:

- Análise de desempenho
- Relatório interno
- Reconciliação da facturação
- Partilha de dados estruturados com clientes

Também garante que os registros de campanha WhatsApp e SMS sejam mantidos separadamente para melhor clareza e organização.

## Tipo de relatório WhatsApp

Para gerar um relatório de download, o administrador deve usar o **Relatório de Pedido** Opção.

Ao solicitar um relatório, o administrador/usuário pode selecionar entre os seguintes tipos de relatório:

### 1. Relatório da Campanha

O Relatório de Campanha inclui detalhes das campanhas iniciadas a partir da aplicação.

O presente relatório contém informações como:

- Número de telemóvel
- Detalhes do conector
- Conteúdo da mensagem
- ID da mensagem
- Estado da mensagem

Este relatório é normalmente usado para rastreamento de entrega e reconciliação.

### 2. Relatório de Botão

Se uma campanha incluir botões interativos (como o Opt-Out ou outros botões de ação), o Relatório de Botão captura interações do destinatário.

O presente relatório inclui:

- Número de beneficiário
- Tempo de interacção
- Botão clicado pelo destinatário

Ajuda os administradores a analisar o comportamento de engajamento e resposta do usuário.

### 3. Relatório de resposta

Se os destinatários responderem a uma mensagem do WhatsApp, essas respostas podem ser baixadas usando a opção Relatório de Resposta.

Este relatório capta:

- Número de beneficiário
- Responder conteúdo
- Data e hora da resposta

É útil para revisar as respostas dos clientes e manter registros de comunicação.

---

## Passos para Baixar um relatório WhatsApp

Siga os passos abaixo para gerar e baixar o relatório necessário:

**Passo 1:** Clique em **Relatório de Pedido**
**Passo 2:** Selecione o desejado **Tipo de relatório**
**Passo 3:** Digite um nome amigável para o relatório
**Passo 4:** Selecione o nome de usuário para o qual o relatório está sendo solicitado
**Passo 5:** Indique o intervalo de datas para obter o relatório
**Passo 6:** Selecione o conector
**Passo 7:** Clique em **Relatório de Pedido** apresentar

![Request Report Dialog](images/downloadwhatsapp2.png)

### Após solicitação

Uma vez solicitado:

- O sistema irá gerar o arquivo em segundo plano.
- A entrada do relatório será visível na UI.
- Aguarde o processamento para completar e atualizar a página, se necessário.
- Quando o estado muda para **Transferir**, o arquivo estará disponível.
- Clique **Transferir** para salvar o relatório no seu sistema local.

---

## Resumo

O módulo WhatsApp Reporting fornece insights de nível sumário e análise detalhada de nível de mensagem dentro de um framework de relatórios unificado.

Com opções como Contagem enviada, Relatório enviado e relatórios de campanha, botão e resposta para download, os administradores podem monitorar efetivamente o desempenho, rastrear interações e manter registros de comunicação estruturados em campanhas do WhatsApp.
