
# Télécharger le rapport WhatsApp

Les **Télécharger le rapport WhatsApp** fonctionnalité permet aux administrateurs de générer et télécharger des rapports de campagne WhatsApp détaillés au format de fichier.

Cette fonctionnalité aide à :

- Analyse des résultats
- Rapports internes
- Réconciliation des factures
- Partage de données structurées avec les clients

Il veille également à ce que les dossiers de campagne de WhatsApp et de SMS soient tenus séparément pour plus de clarté et d'organisation.

## Type de rapport WhatsApp

Pour générer un rapport téléchargeable, l'administrateur doit utiliser **Rapport de demande** option.

Lorsque vous demandez un rapport, l'administrateur/utilisateur peut choisir parmi les types de rapport suivants :

### 1. Rapport de campagne

Le rapport de campagne contient des détails sur les campagnes lancées à partir de la demande.

Ce rapport contient des informations telles que:

- Numéro mobile
- Détails du connecteur
- Contenu du message
- Numéro de message
- État du message

Le présent rapport est habituellement utilisé pour le suivi et le rapprochement de l'exécution.

### 2. Rapport sur le bouton

Si une campagne comprend des boutons interactifs (comme Opt-Out ou d'autres boutons d'action), le rapport de bouton capture les interactions de destinataire.

Ce rapport comprend :

- Numéro du bénéficiaire
- Heure d' interaction
- Bouton cliqué par le destinataire

Il aide les administrateurs à analyser l'engagement des utilisateurs et le comportement de réponse.

### 3. Rapport de réponse

Si les destinataires répondent à un message WhatsApp, ces réponses peuvent être téléchargées en utilisant l'option Reply Report.

Ce rapport présente :

- Numéro du bénéficiaire
- Contenu des réponses
- Date et heure de la réponse

Il est utile pour examiner les réponses des clients et tenir des dossiers de communication.

---

## Étapes pour télécharger un rapport WhatsApp

Suivez les étapes ci-dessous pour générer et télécharger le rapport requis:

**Étape 1:** Cliquez sur **Rapport de demande**
**Étape 2:** Sélectionner le produit désiré **Type de rapport**
**Étape 3:** Saisissez un nom amical pour le rapport
**Étape 4:** Sélectionnez le nom d'utilisateur pour lequel le rapport est demandé
**Étape 5 :** Saisissez la plage de date pour récupérer le rapport
**Étape 6 :** Sélectionnez le connecteur
**Étape 7 :** Cliquez sur **Rapport de demande** à soumettre

![Request Report Dialog](images/downloadwhatsapp2.png)

### Après la demande

Sur demande:

- Le système générera le fichier en arrière-plan.
- L'entrée du rapport sera visible dans l'assurance-chômage.
- Attendez le traitement pour compléter et rafraîchir la page si nécessaire.
- Lorsque le statut change **Télécharger**, le fichier sera disponible.
- Cliquez sur **Télécharger** pour enregistrer le rapport sur votre système local.

---

## Résumé

Le module WhatsApp Reporting fournit à la fois des informations de synthèse et une analyse détaillée des messages dans un cadre de rapport unifié.

Avec des options telles que Sent Count, Sent Report et des rapports de campagne téléchargeables, Button et Reply, les administrateurs peuvent surveiller efficacement les performances, suivre les interactions et maintenir des enregistrements de communication structurés dans toutes les campagnes WhatsApp.
