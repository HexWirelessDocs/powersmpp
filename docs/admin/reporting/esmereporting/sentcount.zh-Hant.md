
# ESME 報告 - 傳送數

這個 **iTextPRO ESME 傳送數** 特性提供 **訊息狀態計數** 每個 **ESME 賬戶** 在給定使用者下。 這有助於詳細分析每個無害環境管理實體的簡訊流量。

## 訪問 SENE 傳送的計數
1. 登入到 **電子文字PRO** 平臺。
2. 地點 **"ESME 傳送數"** 節。

## 檢視無害環境管理報告
- 輸入 **使用者名稱** 在搜尋框中。
- 選擇想要的 **日期範圍** (c) 參閱無害環境管理專家組的報告。

## 瞭解ESME 傳送數
- 顯示 **訊息狀態計數** 每個ESME賬戶。
- **說明:** 這個 **德國航天中心賠償計數** 實值 **沒有** 計入總使用量。

## 每月摘要
- 當a **特定月份** 顯示 **簡訊總數** 那個月。
- A 類 **超連結** 提供訪問 **按日期分列的統計摘要** 用於選定月份。

---

透過使用 **ESME 傳送數** 功能,使用者可以監測ESME特定效能,並獲得對短訊息傳送模式的寶貴見解.
