
# Rapports de l'ESME – Décompte des envois

Les **iTextPRO ESME Compte d'envoi** fonctionnalité fournit une **nombre d'états du message** pour chaque **Compte ESME** sous un utilisateur donné. Cela permet une analyse détaillée du trafic SMS par ESME.

## Accès au compte d'envoi ESME
1. Connectez-vous à la **iTextPRO** plate-forme.
2. Localiser la **"Compte envoyé par l'ESME"** Chapitre.

## Affichage du rapport ESME
- Saisissez la **Nom d'utilisateur** dans la boîte de recherche.
- Sélectionner le produit désiré **date** voir le rapport ESME.

## Comprendre le nombre d'envois d'ESME
- Affichages **nombre d'états du message** pour chaque compte ESME.
- **Remarque:** Les **Nombre d ' indemnités versées par les DLR** est **pas** inclus dans le nombre total d'utilisations.

## Résumé mensuel
- Lorsque **mois spécifique** est sélectionné, iTextPRO affiche **Nombre total de SMS** pour ce mois-là.
- A **Lien hypertexte** est fourni à l'accès **statistiques résumées par date** pour le mois sélectionné.

---

En utilisant **ESME Détachement** fonctionnalité, les utilisateurs peuvent surveiller les performances spécifiques à ESME et obtenir des informations précieuses sur les modes de livraison de SMS.
