
# ESME 报告 - 发送数

这个 **iTextPRO ESME 发送数** 特性提供 **消息状态计数** 每个 **ESME 账户** 在给定用户下。 这有助于详细分析每个无害环境管理实体的短信流量。

## 访问 SENE 发送的计数
1. 登录到 **电子文本PRO** 平台。
2. 地点 **"ESME 发送数"** 节。

## 查看无害环境管理报告
- 输入 **用户名** 在搜索框中。
- 选择想要的 **日期范围** (c) 参阅无害环境管理专家组的报告。

## 了解ESME 发送数
- 显示 **消息状态计数** 每个ESME账户。
- **说明:** 这个 **德国航天中心赔偿计数** 实值 **没有** 计入总使用量。

## 每月摘要
- 当a **特定月份** 显示 **短信总数** 那个月。
- A 类 **超链接** 提供访问 **按日期分列的统计摘要** 用于选定月份。

---

通过使用 **ESME 发送数** 功能,用户可以监测ESME特定性能,并获得对短消息发送模式的宝贵见解.
