
# ESME Reporting – Conde enviado

El **iTextPRO ESME Sent Count** característica proporciona una **recuento de estado del mensaje** para cada uno **Cuenta ESME** bajo un usuario dado. Esto permite un análisis detallado del tráfico de SMS por ESME.

## Accessing ESME Sent Count
1. Inicie sesión en **iTextPRO** plataforma.
2. Localizar el **"Conde de PEME"** sección.

## Viewing ESME Report
- Entra en **nombre de usuario** en la caja de búsqueda.
- Seleccione el deseado **rango de fecha** para ver el informe ESME.

## Entender el Conde enviado de ESME
- Pantallas **estado del mensaje cuenta** para cada cuenta ESME.
- **Nota:** El **DLR compensation count** es **no** incluido en el recuento total de uso.

## Resumen
- Cuando una **mes específico** se selecciona, iTextPRO muestra el **total de SMS** para ese mes.
- A **hipervínculo** se proporciona acceso **estadísticas resumidas correspondientes a la fecha** para el mes seleccionado.

---

Usando el **ESME Sent Count** característica, los usuarios pueden supervisar el rendimiento específico de ESME y obtener información valiosa sobre los patrones de entrega de SMS.
