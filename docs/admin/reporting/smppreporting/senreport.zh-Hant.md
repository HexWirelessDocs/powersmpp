
# 使用者傳送報告

![User Sent Report Overview](images/usersentreport1.png) 
![User Sent Report Details](images/usersentreport2.png)

這個 **傳送的 iTextPRO 報告** 功能可詳細瞭解成功提交的簡訊流量,包括髮送狀況和相關細節。

## 使用者專用搜索
- 使用者可以輸入特定的 **登入** 輸入搜尋框以訪問特定使用者的傳送記錄。
- 選擇想要的 **日期** 縮小搜尋範圍並檢索相關資訊。

## 金鑰交付記錄細節
- **提交日期** – 表示使用者將信件傳送到 iTextPRO 的時間 。
- **移動號碼** 目標運動的目標地址。
- **發件人標識** - 競選所使用的發端人地址或源地址。
- **訊息** - 使用者傳送的信件正文。
- **狀態** - 供應商閘道器收到交貨報告。
- **短訊息費用** ——使用者的短訊息成本價格,以基準貨幣顯示.
- **閘道器標識** – 識別訊息基於所配置的路由路由的供應商閘道器.
- **錯誤程式碼** - 供應商閘道器收到錯誤程式碼。

## 快速訪問和建議
- 為了取得更快的結果,請選擇一個 **較短的日期範圍** 建議。

## 詳細信件資訊
點選信件內容可以發現更多細節,包括:
- **訊息編號** - 獲得承認後由門戶提供者/供應商提供。 
 在 DLR 補償中,資訊編號包括字首 **老師**。 。 。 。
- **別名信件ID** – 系統生成訊息- id.

## 檢視信件連結
- 使用者可以單擊 **"影片訊息"** 連結以使用者傳送的實際格式顯示信件內容。
