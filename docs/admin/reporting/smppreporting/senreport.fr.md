
# Rapport envoyé par l'utilisateur

![User Sent Report Overview](images/usersentreport1.png) 
![User Sent Report Details](images/usersentreport2.png)

Les **iTextPRO a envoyé des rapports** fonctionnalité fournit des informations détaillées sur le trafic SMS soumis avec succès, y compris l'état de livraison et les détails pertinents.

## Recherche axée sur l'utilisateur
- Les utilisateurs peuvent saisir une donnée **Connexion** dans la boîte de recherche pour accéder aux dossiers de livraison d'un utilisateur particulier.
- Choisissez le produit désiré **date période** de restreindre la recherche et de récupérer l'information pertinente.

## Principaux détails du dossier de livraison
- **Date de présentation** – Indique quand l'utilisateur a envoyé le message à iTextPRO.
- **Numéro mobile** – Adresse de destination pour la campagne ciblée.
- **ID de l'expéditeur** – Adresse d'origine ou adresse source utilisée pour la campagne.
- **Message** – Corps du message envoyé par l'utilisateur.
- **État** – Rapport de livraison reçu par la passerelle du fournisseur.
- **Coût du SMS** – prix SMS pour l'utilisateur, visible dans la monnaie de base.
- **ID de la passerelle** – Indique la passerelle du fournisseur vers laquelle le message est acheminé en fonction du routage configuré.
- **Code d'erreur** – Code d'erreur reçu par la passerelle du fournisseur.

## Accès rapide et recommandations
- Pour des résultats plus rapides, sélectionner un **durée plus courte** est recommandé.

## Informations détaillées sur le message
Cliquer sur le contenu du message révèle des détails supplémentaires, notamment :
- **Message-ID** – Fourni par le fournisseur de passerelle / Vendeur sur reconnaissance. 
 Dans la compensation DLR, le message-id inclut le préfixe **Les**.
- **Alias Message-ID** – Message-id généré par le système.

## Afficher le lien du message
- Les utilisateurs peuvent cliquer sur **"Voir le message"** lien pour afficher le contenu du message dans le format réel envoyé par l'utilisateur.

---

Les **iTextPRO a envoyé des rapports** fonctionnalité permet aux utilisateurs avec des informations complètes sur le trafic SMS, leur permettant de **suivre les livraisons, les problèmes de dépannage et les détails du message d'examen** sans effort.
