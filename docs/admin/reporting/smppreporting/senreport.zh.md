
# 用户发送报告

![User Sent Report Overview](images/usersentreport1.png) 
![User Sent Report Details](images/usersentreport2.png)

这个 **发送的 iTextPRO 报告** 功能可详细了解成功提交的短信流量,包括发送状况和相关细节。

## 用户专用搜索
- 用户可以输入特定的 **登录** 输入搜索框以访问特定用户的发送记录。
- 选择想要的 **日期** 缩小搜索范围并检索相关信息。

## 密钥交付记录细节
- **提交日期** – 表示用户将信件发送到 iTextPRO 的时间 。
- **移动号码** 目标运动的目标地址。
- **发件人标识** - 竞选所使用的发端人地址或源地址。
- **消息** - 用户发送的信件正文。
- **状态** - 供应商网关收到交货报告。
- **短消息费用** ——用户的短消息成本价格,以基准货币显示.
- **网关标识** – 识别消息基于所配置的路由路由的供应商网关.
- **错误代码** - 供应商网关收到错误代码。

## 快速访问和建议
- 为了取得更快的结果,请选择一个 **较短的日期范围** 建议。

## 详细信件信息
点击信件内容可以发现更多细节,包括:
- **消息编号** - 获得承认后由门户提供者/供应商提供。 
 在 DLR 补偿中,信息编号包括前缀 **老师**。 。 。 。
- **别名信件ID** – 系统生成消息- id.

## 查看信件链接
- 用户可以单击 **"视频消息"** 链接以用户发送的实际格式显示信件内容。
