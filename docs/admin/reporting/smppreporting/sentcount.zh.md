
# 用户发送数

![User Sent Counts](images/usersentcount1.png)

这个 **iTextPRO 发送数** 特性为单一用户提供了一个消息状态计数,从而能够全面跟踪短信的性能。

## 用户特定搜索
- 输入所需 **用户名** 输入搜索框。
- 选择需要的 **日期范围** 访问用户消息的状态计数报告。

## DLR 赔偿说明
- 这个 **消费总量** 显示确实 **没有** 包括德国航天中心的赔偿点数。

## 每月摘要显示
- 选择a **月份** 将默认显示全部短消息数。
- A 类 **超链接** ,允许用户钻入更详细的 **按日期分列的统计摘要** 那个月。

## 时区审议
- 显示的消息状态计数基于 **用户的时区**。 。 。 。

---

与 **iTextPRO 发送的计数** 特性,用户可以轻松 **监视信件状态**,评估性能,并探索详细统计,以更深入地了解短信流量.
