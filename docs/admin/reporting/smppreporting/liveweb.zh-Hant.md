# 報告

## iTextPRO 報告模組概覽

這個 **iTextPRO 報告模組** 在評估簡訊流量的準確性和及時性方面發揮關鍵作用。 
主要特點和功能包括:

### 交付報告引擎
- 利用複雜的快取機制 **100%的交貨收據(DLR)** 捕獲。
- 實現全面狀態報告,提供資訊傳遞的深度資訊.

### 與執行引擎同步
- 路由引擎和iTextPRO交付報告引擎之間的隱形同步.
- 便利 **適應性路線** 基於 **服務質量(質量)** 最佳效能和高效傳遞資訊。

### QoS- Driven 適應性路線
- 基於實時服務質量計量的動態路由調整.
- 確保傳送短訊息的最大效能和有效性.

### 準確而及時的評估
- 使管理員能夠評估簡訊流量的準確性和及時性.
- 透過全面報告詳細瞭解簡訊的效能。

---

整合這些功能可確保iTextPRO不僅提供可靠的報告能力,而且使路由決定與服務質量衡量標準保持一致。 
這種協同作用有助於增強業績,有效傳遞資訊,並更深入地瞭解簡訊的交通環境。

## 現場網路交通報告

![Live Web Traffic Report](images/liveweb1.png)

這個 **現場網路交通報告** 在iTextPRO中,對透過網路介面傳輸的簡訊流量提供了實時的透視.

### 關鍵特性

#### 實時簡訊交通監測
- 提供來自網路介面的簡訊流量的現場顯示.
- 允許使用者實時監控短訊息的流出.

#### 前十名發件人
- 提供清單 **前十名發件人** 基於交通消費。
- 使使用者能夠識別和評估簡訊流量的最積極貢獻者.

#### 當前賬戶餘額
- 在 **基準貨幣**。 。 。 。
- 向用戶提供其賬戶餘額的最新資訊。

#### 等級結構
- 呈現與使用者賬戶相連結的父賬戶的分級結構.
- 明確反映不同賬戶之間的關係。

#### 綜合資訊
- 確保使用者能夠獲得關於簡訊流量、傳送者活動、賬戶餘額和賬戶關係的完整資訊。
- 協助a **全面理解** (原始內容存檔於2019-09-21). SMS交通景觀.

---

"Live Web Traffic Report"將使用者的能見度提升到實時簡訊活動,傳送者行為,以及賬戶動態. 
它是一個寶貴的工具 **監測、分析和決策** 在 iTextPRO 平臺內。
