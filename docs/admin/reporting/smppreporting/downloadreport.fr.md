
# Télécharger le rapport

Les **iTextPRO Télécharger le rapport** fonctionnalité permet aux utilisateurs de télécharger facilement des rapports SMS dans **Format Excel** pour une analyse détaillée.

![Download Report Option](images/downloadreport1.png)

## Accès à la demande de téléchargement
1. Naviguez vers la **iTextPRO** plate-forme.
2. Localiser la **"Rapport de téléchargement"** option.

## Ouverture de la demande de téléchargement
- Cliquez sur le **"Rapport de demande"** option.
- Une page contextuelle apparaîtra pour saisir les détails du téléchargement.

![Download Report Request](images/downloadreport2.png)

## Saisir les détails du téléchargement
- **Titre du rapport** – Fournir un nom convivial pour le rapport.
- **Nom d'utilisateur** – Spécifiez le nom d'utilisateur du compte utilisateur du rapport.
- **Gamme de dates** – Définir la période pour le rapport. *(Note : une plus longue plage de dates peut prendre plus de temps à traiter.)*
- **ID de l'expéditeur** *(facultatif)* – Inclure un identifiant d'expéditeur spécifique si nécessaire.

## Suivi des demandes de téléchargement
- Après avoir soumis la demande, iTextPRO affiche une liste des demandes de téléchargement existantes ainsi que leur **État actuel**.
- Les utilisateurs peuvent suivre l'évolution de leurs demandes de téléchargement.

## Téléchargement des rapports achevés
![Download Report Completed](images/downloadreport3.png) 
- Une fois que l'état du rapport indique **Achèvement**, les utilisateurs peuvent le télécharger en cliquant sur **bouton de téléchargement**.

---

Avec **Rapport de téléchargement de iTextPRO** fonctionnalité, les utilisateurs peuvent facilement **demander, suivre et télécharger** rapports SMS spécifiques aux utilisateurs pour une analyse complète.
