
# Relatório Enviado pelo Usuário

![User Sent Report Overview](images/usersentreport1.png) 
![User Sent Report Details](images/usersentreport2.png)

A **Relatórios enviados pelo iTextPRO** O recurso fornece informações detalhadas sobre o tráfego SMS enviado com sucesso, incluindo status de entrega e detalhes relevantes.

## Pesquisa Focada pelo Usuário
- Os utilizadores podem introduzir uma **login** na caixa de pesquisa para acessar os registros de entrega de um determinado usuário.
- Escolha o desejado **período de data** Reduzir a busca e recuperar informações relevantes.

## Detalhes do registro da entrega chave
- **Data da apresentação** – Indica quando o usuário enviou a mensagem para o iTextPRO.
- **Número Móvel** – Endereço de destino para a campanha direcionada.
- **ID do remetente** – Endereço Originador ou Endereço Fonte usado para a campanha.
- **Mensagem** – Corpo da mensagem enviada pelo usuário.
- **Estado** – Relatório de entrega recebido pelo gateway do fornecedor.
- **Custo de SMS** – Preço de custo SMS para o usuário, visível na moeda base.
- **ID da porta** – Identifica o gateway do fornecedor para o qual a mensagem é roteada com base no roteamento configurado.
- **Código de Erro** – Código de erro recebido pelo gateway do fornecedor.

## Acesso rápido e recomendações
- Para resultados mais rápidos, selecionar um **intervalo de datas mais curto** é recomendado.

## Informação detalhada da mensagem
Clicar no conteúdo da mensagem revela detalhes adicionais, incluindo:
- **ID da mensagem** – Providenciado pelo Provedor de Gateway / Vendor após reconhecimento. 
 Na compensação DLR, o ID da mensagem inclui o prefixo **MR**.
- **ID da Mensagem** - Mensagem gerada pelo sistema.

## Ver a Ligação da Mensagem
- Os usuários podem clicar no **"Ver Mensagem"** link para exibir o conteúdo da mensagem no formato real enviado pelo usuário.

---

A **Relatórios enviados pelo iTextPRO** recurso capacita usuários com insights abrangentes sobre o tráfego de SMS, permitindo-lhes **rastrear entregas, problemas de solução de problemas e analisar detalhes da mensagem** Sem esforço.
