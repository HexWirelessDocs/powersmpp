
# 使用者傳送數

![User Sent Counts](images/usersentcount1.png)

這個 **iTextPRO 傳送數** 特性為單一使用者提供了一個訊息狀態計數,從而能夠全面跟蹤簡訊的效能。

## 使用者特定搜尋
- 輸入所需 **使用者名稱** 輸入搜尋框。
- 選擇需要的 **日期範圍** 訪問使用者訊息的狀態計數報告。

## DLR 賠償說明
- 這個 **消費總量** 顯示確實 **沒有** 包括德國航天中心的賠償點數。

## 每月摘要顯示
- 選擇a **月份** 將預設顯示全部短訊息數。
- A 類 **超連結** ,允許使用者鑽入更詳細的 **按日期分列的統計摘要** 那個月。

## 時區審議
- 顯示的訊息狀態計數基於 **使用者的時區**。 。 。 。

---

與 **iTextPRO 傳送的計數** 特性,使用者可以輕鬆 **監視信件狀態**,評估效能,並探索詳細統計,以更深入地瞭解簡訊流量.
