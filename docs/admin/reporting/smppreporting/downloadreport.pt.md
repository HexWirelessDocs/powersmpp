
# Baixar relatório

A **Relatório de Download iTextPRO** recurso permite que os usuários baixem facilmente relatórios SMS em **Formato Excel** para análise detalhada.

![Download Report Option](images/downloadreport1.png)

## Acessando pedido de download
1. Navegar para o **iTextPRO** plataforma.
2. Localizar o **"Relatório de download"** Opção.

## Iniciando pedido de download
- Clique na **"Relatório de Pedido"** Opção.
- Uma página pop-up aparecerá para inserir detalhes de download.

![Download Report Request](images/downloadreport2.png)

## A introduzir os detalhes da transferência
- **Título do relatório** – Fornecer um nome amigável para o relatório.
- **Utilizador** – Especifique o nome de usuário da conta do usuário para o relatório.
- **Intervalo de Datas** – Defina o período para o relatório. *(Nota: Um intervalo de datas mais longo pode levar mais tempo para processar.)*
- **ID do remetente** *(Opcional)* – Inclua um identificador de remetente específico, se necessário.

## Rastreando pedidos de download
- Ao enviar o pedido, o iTextPRO exibe uma lista de pedidos de download existentes juntamente com seus **estado atual**.
- Os usuários podem monitorar o progresso de suas solicitações de download.

## Baixando Relatórios Concluídos
![Download Report Completed](images/downloadreport3.png) 
- Assim que o estado do relatório indicar **completar**, os usuários podem baixá-lo clicando no **botão de download**.

---

Com **Relatório de Download do iTextPRO** recurso, os usuários podem convenientemente **request, track, and download** Relatórios SMS específicos do utilizador para uma análise exaustiva.
