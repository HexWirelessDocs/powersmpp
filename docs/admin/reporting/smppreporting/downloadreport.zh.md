
# 下载报表

这个 **iTextPRO 下载报告** 特性允许用户在 **Excel 格式** 详细分析。

![Download Report Option](images/downloadreport1.png)

## 访问下载请求
1. 导航到 **电子文本PRO** 平台。
2. 地点 **"下载报告"** 选项。

## 启动下载请求
- 点击 **"请求报告"** 选项。
- 将会出现一个弹出页面,用于输入下载细节 。

![Download Report Request](images/downloadreport2.png)

## 输入下载细节
- **报告标题** - 为报告提供方便用户的名称。
- **用户名** - 指定报表用户账户的用户名。
- **日期范围** - 确定报告期限。 *(注:较长的日期范围可能需要更多的时间处理。 )*
- **发件人标识** *(备选)* - 如果需要,包括特定的发送者身份。

## 跟踪下载请求
- 提交请求时, iTextPRO 将显示一份现有下载请求清单及其 **现况**。 。 。 。
- 用户可以监视其下载请求的进展情况.

## 下载已完成的报告
![Download Report Completed](images/downloadreport3.png) 
- 一旦报告状况表明 **完成**,用户可以点击 **下载按钮**。 。 。 。

---

与 **iTextPRO 的下载报告** 特性,用户可以方便 **请求、跟踪和下载** 用户专用短信报告,供全面分析。
