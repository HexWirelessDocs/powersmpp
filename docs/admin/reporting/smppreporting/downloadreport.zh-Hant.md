
# 下載報表

這個 **iTextPRO 下載報告** 特性允許使用者在 **Excel 格式** 詳細分析。

![Download Report Option](images/downloadreport1.png)

## 訪問下載請求
1. 導航到 **電子文字PRO** 平臺。
2. 地點 **"下載報告"** 選項。

## 啟動下載請求
- 點選 **"請求報告"** 選項。
- 將會出現一個彈出頁面,用於輸入下載細節 。

![Download Report Request](images/downloadreport2.png)

## 輸入下載細節
- **報告標題** - 為報告提供方便使用者的名稱。
- **使用者名稱** - 指定報表使用者賬戶的使用者名稱。
- **日期範圍** - 確定報告期限。 *(注:較長的日期範圍可能需要更多的時間處理。 )*
- **發件人標識** *(備選)* - 如果需要,包括特定的傳送者身份。

## 跟蹤下載請求
- 提交請求時, iTextPRO 將顯示一份現有下載請求清單及其 **現況**。 。 。 。
- 使用者可以監視其下載請求的進展情況.

## 下載已完成的報告
![Download Report Completed](images/downloadreport3.png) 
- 一旦報告狀況表明 **完成**,使用者可以點選 **下載按鈕**。 。 。 。

---

與 **iTextPRO 的下載報告** 特性,使用者可以方便 **請求、跟蹤和下載** 使用者專用簡訊報告,供全面分析。
