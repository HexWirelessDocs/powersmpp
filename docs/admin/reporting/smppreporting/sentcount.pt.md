
# Contagens enviadas pelo usuário

![User Sent Counts](images/usersentcount1.png)

A **Contagem enviada pelo iTextPRO** O recurso fornece uma contagem de status de mensagem para um único usuário, permitindo o rastreamento abrangente do desempenho de SMS.

## Pesquisa Específica do Usuário
- Introduzir o desejado **utilizador** na caixa de pesquisa.
- Selecione o necessário **intervalo de datas** para acessar o relatório de contagem de status da mensagem do usuário.

## Nota de compensação DLR
- A **número total de consumos** exibido faz **não** Incluir a contagem de compensação DLR.

## Apresentação mensal do resumo
- Escolher um **mês** irá mostrar a contagem total de SMS por padrão.
- A **hiperlink** é fornecido, permitindo que os usuários para perfurar para baixo em mais detalhado **estatísticas sumárias date-wise** para esse mês.

## Considerações sobre o fuso horário
- A contagem do estado da mensagem exibida é baseada na **fuso horário do usuário**.

---

Com **Contagem enviada do iTextPRO** recurso, os usuários podem facilmente **estado da mensagem monitor**, avaliar o desempenho e explorar estatísticas detalhadas para uma compreensão mais profunda do tráfego de SMS.
