# 网关智慧计数

**导航 :** <span data-ph="0"></span> ➔ <span data-ph="1"></span> ➔ <span data-ph="2"></span>。 。 。 。

## 概览

这个 **网关智慧计数** 报告为管理员提供了所有已配置网关上信件提交量的综合视图。 报告涵盖 **过去7天** 每日一次,便于一目了然地确定交通模式和载荷分布。

![Gateway Wise Count — All Gateways (7-day view)](images/gwcount-01-all-gateways.png)

---

## 使用网关过滤器

要将报告缩小到特定的网关,请从 **下拉过滤器** 可在报告屏幕上找到。 该表将刷新仅显示该网关的提交数据。

![Gateway Wise Count — Filtered by a single gateway](images/gwcount-02-filtered.png)

---

## 报告显示的内容

| 栏 | 说明 |
|--------|-------------|
| **网关名称** | 配置的网关的友好名称 。 |
| **日期栏(每日一栏)** | 在相应的日期通过该网关发送了信件总数。 |

!!! note
 可见信件计数按每个计算 **网关自己的时区**,而不是应用程序时区。 表上页注示了这一点。

---

## 何时使用此报表

- 一天到一天的零星流量下降,可能表明供应商或网络问题。
- 确定未充分利用的网关,这些网关可能是退休或改道的候选人。
- 验证新添加的网关是否收到预期流量份额 。
- 为利益攸关方提供快速的每周亮相快照。
