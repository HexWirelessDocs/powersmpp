
# Relatório do Estado do Gateway

iTextPRO oferece **Relatório de Estado de Gateway abrangente**, permitindo que os usuários rapidamente **avaliar e comparar** O desempenho de vários gateways de fornecedores.

![Gateway Performance](images/gatewayperformance1.png)

## Categorias de Desempenho

### Melhor Realização
- **Percentagem de entrega:** Terminado **80%**
- Indicações **desempenho de entrega excepcional**.

### Boa execução
- **Percentagem de entrega:** Terminado **60%**
- Representa **desempenho de entrega louvável**.

### Realização Média
- **Percentagem de entrega:** Terminado **40%**
- Refletir **desempenho de entrega moderado**.

---

## Acesso ao Relatório de Estado do Gateway

1. Digite o **nome do gateway do fornecedor** na caixa de pesquisa. 
2. Selecione o desejado **intervalo de datas**. 
3. Enviar a pesquisa para gerar um **Relatório de desempenho pormenorizado** para o gateway selecionado. 

---

## Notas Importantes
- A **contagem do estado da mensagem** no relatório é exibido de acordo com o **fuso horário do usuário**.

---

A **Relatório do Estado do Gateway** capacita os administradores a:
- **Comparar** desempenho do gateway do fornecedor.
- **Identificar** Gateways de alto desempenho.
- **Otimizar estratégias de entrega SMS** baseado em métricas de desempenho precisas.
