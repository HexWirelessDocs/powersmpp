# 网关智能利润报告

**导航 :** <span data-ph="0"></span> ➔ <span data-ph="1"></span> ➔ <span data-ph="2"></span>。 。 。 。

## 概览

这个 **网关智能利润报告** 为管理员提供了网关级利润率和短信流量分析的详细分类. 其目的是支持财务分析、计费核查和网关业绩监测,方法是: **购买费用**, (中文(简体) ). **销售价值**,以及 **利润幅度** 给每个网关的短消息

---

从下放中选择一个特定的网关,以生成一个覆盖到该网关流量的利润报告.

## 2. 国家选择

![Gateway Wise Profit — Country filter dropdown](images/gwprofit-02-country-filter.png)

## 3. 日期范围过滤器

管理员可以定义 **自定义日期范围** 以生成任何历史时期的利润报告。

!!! info "日期范围特征"
    - 每日和历史交通分析
    - 基于日期的灵活过滤
    - 计费期核查支助
    - 任何时间范围内的业务业绩审查

## 4. 表视图报告

这个 **表格视图** 以结构化表格格式列出国别网关利润数据,并列如下:

| 栏 | 说明 |
|--------|-------------|
| **网关名称** | 网关通车. |
| **国家名称** | 交通目的地国. |

## 6. 利润计算公式

报告采用下列标准公式得出利润度量:

```
Margin (Base Currency) =  Sales Price − Purchase Price

Margin Percentage (%)  = ((Sales Price − Purchase Price) / Purchase Price) × 100
```

## 网关明智利润报告的目的

!!! info "用这个报告来..."
    - 实时监测网关盈利情况
    - 按国家和网关比值
    - 跟踪短信流量和相关费用
    - 审查网关收入与购买费用
    - 支持财务报告和账单核查
    - 优化路线战略以更好地管理收入
    - 用于离线分析和记录保存的出口报告
