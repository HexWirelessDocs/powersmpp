
# 网关状态报告

iTextPRO提供 **全面门户状况报告**,允许用户快速 **评估和比较** 各种供应商网关的业绩。

![Gateway Performance](images/gatewayperformance1.png)

## 业绩类别

### 最佳表演
- **交付百分比 :** 完毕 **80%**
- 说明 **特殊交付业绩**。 。 。 。

### 表现良好
- **交付百分比 :** 完毕 **60%**
- 数额 **执行业绩值得赞扬**。 。 。 。

### 平均成绩
- **交付百分比 :** 完毕 **40%**
- 反映 **中度交付业绩**。 。 。 。

---

## 访问网关状态报告

1. 输入 **供应商网关名称** 在搜索框中。 
2. 选择想要的 **日期范围**。 。 。 。 
3. 提交搜索以生成 **详细执行情况报告** 用于选定网关。 

---

## 重要说明
- 这个 **消息状态计数** 在报表中按照 **用户的时区**。 。 。 。

- **识别** 顶级的网关
- **优化短信传送战略** 根据准确的业绩衡量标准。
