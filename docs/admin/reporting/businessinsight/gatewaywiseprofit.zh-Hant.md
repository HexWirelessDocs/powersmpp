# 閘道器智慧利潤報告

**導航 :** <span data-ph="0"></span> ➔ <span data-ph="1"></span> ➔ <span data-ph="2"></span>。 。 。 。

## 概覽

這個 **閘道器智慧利潤報告** 為管理員提供了閘道器級利潤率和簡訊流量分析的詳細分類. 其目的是支援財務分析、計費核查和閘道器業績監測,方法是: **購買費用**, (中文(簡體) ). **銷售價值**,以及 **利潤幅度** 給每個閘道器的短訊息

---

## 1. 選擇閘道器

從下放中選擇一個特定的閘道器,以生成一個覆蓋到該閘道器流量的利潤報告.

![Gateway Wise Profit — Gateway dropdown selection](images/gwprofit-01-gateway-dropdown.png)

---

## 2. 國家選擇

該報告可由目的地國進一步過濾,從而能夠在國家一級進行顆粒性利潤分析。

![Gateway Wise Profit — Country filter dropdown](images/gwprofit-02-country-filter.png)

---

## 3. 日期範圍過濾器

管理員可以定義 **自定義日期範圍** 以生成任何歷史時期的利潤報告。

!!! info "日期範圍特徵"
    - 每日和歷史交通分析
    - 基於日期的靈活過濾
    - 計費期核查支助
    - 任何時間範圍內的業務業績審查

---

## 4. 表檢視報告

這個 **表格檢視** 以結構化表格格式列出國別閘道器利潤資料,並列如下:

| 欄 | 說明 |
|--------|-------------|
| **閘道器名稱** | 閘道器通車. |
| **國家名稱** | 交通目的地國. |
| **MCCMNC 中超聯賽** | 移動鄉村程式碼+行動網路程式碼. |
| **網路名稱** | 目的地行動網路. |
| **信件計數** | 此切片已全部發送短訊息 。 |
| **購買價格** | 這片菜要花很多錢 |
| **銷售價格** | 收入來自這個片段。 |
| **邊際(銷售-購買)%** | 計算利潤百分比。 |
| **邊際(銷售-購買)(EURO)** | 歐洲區域辦事處的絕對利潤值。 |

![Gateway Wise Profit — Table View](images/gwprofit-03-table-view.png)

!!! note
 <span data-ph="0"></span> 相對於某一價格,則表明該價格為平價。

## 6. 利潤計算公式

報告採用下列標準公式得出利潤度量:

```
Margin (Base Currency) =  Sales Price − Purchase Price

Margin Percentage (%)  = ((Sales Price − Purchase Price) / Purchase Price) × 100
```

## 閘道器明智利潤報告的目的

!!! info "用這個報告來..."
    - 實時監測閘道器盈利情況
    - 按國家和閘道器比值
    - 跟蹤簡訊流量和相關費用
    - 審查閘道器收入與購買費用
    - 支援財務報告和賬單核查
    - 最佳化路線戰略以更好地管理收入
    - 用於離線分析和記錄儲存的出口報告
