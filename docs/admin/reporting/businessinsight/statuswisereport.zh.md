
# 状态明智报告

iTextPRO提供 **强有力的报告模块**,使用户能够访问 **供应商和用户的详细报告** ——对准确计费和绩效分析至关重要.

---

## 1. 供应商智能交通
- 显示 **供应商流量** 基于 **时区** 每个供应商网关。 
- 计算包括实际提交的信息, **不包括德国航天中心赔偿短消息数**。 。 。 。 
- 提供深入了解 **交付状况** 财务报告和审定财务报表 **每个供应商网关的业绩**。 。 。 。 
- **密钥量度 :** 
  - **战略计划** - 销售价格 
  - **联合国** - 成本价格 
  - **边距** – 说吧 – <span data-ph="0"></span>

![Vendor Wise Traffic](images/statuswise1.png)

## 2. 按国家分列的供应商流量
- 选择一个 **日期范围** 以查看提交给每个国家的短信数。 
- **下沉选项** 显示供应商网关名称和详细的短信信息。 
- 能够对 **短消息流量模式** 内 **具体国家和网络**。 。 。 。

![Country-wise Vendor Traffic](images/statuswise2.png)

## 3. 伙伴智能交通
- 显示 **以伙伴方式传送** 基于 **时区** 每个用户。 
- 显示 **短信流量** 财务报告和审定财务报表 **交付状况** 每一个合伙人。 
- **下沉视图** 显示与用户名有联系的国家和特定网络。

## 4. 明智国家伙伴交通
- 规定 a **详细报告** (b) 合作伙伴向每个国家提交的短信数。

- **下沉视图** 提供深入了解 **通讯活动** 跨越不同国家和网络。

## 5. 供应商明智伙伴报告

这个 **供应商明智伙伴报告**,在 *状况报告* 部分,为管理员提供了网关级流量及其相关用户消耗的综合视图。

本报告旨在帮助管理人员了解整个平台如何使用一个特定网关。

### 报告的目的

本报告的主要目标是:

- 显示通过选定网关处理的全部流量
- 显示通过该网关的用户路径流量列表
- 目前每个用户的消费总量
- 为开单和业务审查提供可见度

这样管理员就可以从一个单一的界面评价网关的性能和使用分布.

![Vendor Wise Partner Report](images/statuswise5.png)

---

## 6. 伙伴明智供应商报告

这个 **伙伴明智供应商报告**,在 *状况报告* ,以用户为中心查看多个网关的流量分布。

本报告重点介绍特定用户的流量如何通过不同的网关进行路由.

### 报告的目的

报告允许管理员:

- 审查选定用户的总使用情况
- 分析流量如何在指定的网关之间分布
- 监测跨多条路线的提交数量
- 检查每个网关的状态发送数

它提供了一个用户的路线行为和消费模式的完整图景.

![Partner Wise Vendor Report](images/statuswise6.png)
