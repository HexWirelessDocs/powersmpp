# 用户智能利润报告

**导航 :** <span data-ph="0"></span> ➔ <span data-ph="1"></span> ➔ <span data-ph="2"></span>。 。 。 。

## 概览

这个 **用户智能利润报告** 使管理员能够分析任何选定日期范围的用户级短信流量和利润率。 报告基于 **完全基于用户通过应用程序提交的信息** 并提供每个客户账户的详细收入、成本和差值数据。

---

## 1. 选择用户

从下拉列表中选择一个特定的用户账户来为该客户生成目标利润报告.

## 2. 国家选择

按目的地国过滤流量数据,以获得每个区域的用户利润率的颗粒分析。

## 3. 日期范围过滤器

定义自定义的日期范围,以生成任何计费或运营期间的用户利润报告.

### 引用栏

| 栏 | 说明 |
|--------|-------------|
| **用户名** | 用户账户的名称/标识符 。 |
| **国家名称** | 用户短信流量的目的地国. |
| **MCCMNC 中超联赛** | 移动乡村代码+移动网络代码. |
| **网络名称** | 目标移动网络名称. |
| **信件计数** | 用户提交的短消息总数. |
| **购买价格** | 办理用户流量实际发生的路由费用. |
| **销售价格** | 向用户收取的销售总额。 |
| **边际百分比(销售-购买)** | 计算用户流量的利润百分比. |
| **边际(销售-购买)** | 从用户流量中获得的总利润(美元)。 |

## 6. 信件提交分析

用户智能利润报告中的所有计算都驱动 **专职** 通过用户端通过应用程序提交的信息。 绕过应用程序用户提交路径的流量不包含在本报告中.

---

## 7. 利润计算公式

```
Margin (USD)          =  Sales Price − Purchase Price

Margin Percentage (%) = ((Sales Price − Purchase Price) / Purchase Price) × 100
```

!!! tip
 将本报告与 **网关智能利润** 从两个互补角度——客户(收益方)和供应商(成本方)——看利润率。
