# 閘道器智慧計數

**導航 :** <span data-ph="0"></span> ➔ <span data-ph="1"></span> ➔ <span data-ph="2"></span>。 。 。 。

## 概覽

這個 **閘道器智慧計數** 報告為管理員提供了所有已配置閘道器上信件提交量的綜合檢視。 報告涵蓋 **過去7天** 每日一次,便於一目瞭然地確定交通模式和載荷分佈。

![Gateway Wise Count — All Gateways (7-day view)](images/gwcount-01-all-gateways.png)

---

## 使用閘道器過濾器

要將報告縮小到特定的閘道器,請從 **下拉過濾器** 可在報告螢幕上找到。 該表將重新整理僅顯示該閘道器的提交資料。

![Gateway Wise Count — Filtered by a single gateway](images/gwcount-02-filtered.png)

---

## 報告顯示的內容

| 欄 | 說明 |
|--------|-------------|
| **閘道器名稱** | 配置的閘道器的友好名稱 。 |
| **日期欄(每日一欄)** | 在相應的日期透過該閘道器傳送了信件總數。 |

!!! note
 可見信件計數按每個計算 **閘道器自己的時區**,而不是應用程式時區。 表上頁注示了這一點。

---

## 何時使用此報表

- 一天到一天的零星流量下降,可能表明供應商或網路問題。
- 確定未充分利用的閘道器,這些閘道器可能是退休或改道的候選人。
- 驗證新新增的閘道器是否收到預期流量份額 。
- 為利益攸關方提供快速的每週亮相快照。
