便利 **现场测试** 在服务器上,您需要共享您的 **SMPP 绑定证书** 页:1 **SMS MO( 移动源代码)** 财务报告和审定财务报表 **MT( 机动性终止)** 连接。 这些全权证书一般取自:

- **短消息网关供应商** 或者说 
- **短消息服务中心( Short Message Service Centre)**

此实体播放 **作用重大** 将短消息存储、转发、转换和传送到移动话筒中——通常以 **电信公司** 或一个 **第三方合计**。 。 。 。

## SMPP 连接参数样本

你那典型的 **电话或短信** 将为你们提供下列基本参数:

| 参数 | 需求 | 说明 |
|-----------|:--------:|-------------|
| **SMSC IP 地址** | :material-check: | SMSC 服务器的IP 地址 |
| **SMSC 端口** | :material-check: | SMPP 连接的端口号 |
| **系统标识** | :material-check: | SMPP 认证的用户名 |
| **密码** | :material-check: | SMPP 认证密码 |
| **系统类型** | | 无害环境管理系统的可选标识符 |
| **绑定模式** | :material-check: | 传送器/接收器/收发器 |
| **届会次数** | :material-check: | Tx / Rx / TxRx 会话计数 |
| **顿** | | 数字类型(例如,国际,字母数字) |
| **NPI 状态** | | 编号计划指标(例如E.164、ISDN) |
