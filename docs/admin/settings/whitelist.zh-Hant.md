
# 白色列表數字

這個 **白色列表數字** 特性允許管理員定義全球白列移動數字列表。 何時 **德國航天中心賠償** 在應用程式中啟用,傳送到白名單所列數字的任何信件將始終提交給供應商, **將不考慮德國航天中心的賠償**,無論交付狀況如何。

## 目的

這一特徵有助於將具體數字(如測試數字、內部監測數字或優先聯絡人)排除在補償邏輯之外,同時確保資訊傳遞不中斷。

![White List Numbers](images/whitelist1.png)

---

## 將數字新增到白名單

1. 導航到 **白色列表數字** 區域。
2. 點選 **新增新內容**。 。 。 。
3. 輸入輸入欄位中的移動數字。
4. 透過用逗號將它們分開,可以同時新增多個數字(<span data-ph="0"></span>) (中文(簡體) ).
5. 確保 **每個手機號碼包括國家程式碼** (如:+91XXXXXXXXXX.
6. 儲存更改以在全球應用白名單條目 。

![Add Whitelist Numbers](images/whitelist2.png)

---

## 刪除列入白名單的數字

應用程式支援 **批次刪除**,允許管理員刪除單個動作中的多個白名單條目。

管理員也可刪除 **單個選定條目** 視需要而定。

![Bulk Delete Whitelist](images/whitelist4.png)

---

## 關鍵說明

!!! info "重要"
    - 採用白名單編號 **整個應用系統**。 。 。 。
    - 傳送到這些數字的信件是 **從德國航天中心賠償計算中排除**。 。 。 。
    - 正確功能必須採用國家程式碼的適當格式。
