
# 白色列表数字

这个 **白色列表数字** 特性允许管理员定义全球白列移动数字列表。 何时 **德国航天中心赔偿** 在应用程序中启用,发送到白名单所列数字的任何信件将始终提交给供应商, **将不考虑德国航天中心的赔偿**,无论交付状况如何。

## 目的

这一特征有助于将具体数字(如测试数字、内部监测数字或优先联系人)排除在补偿逻辑之外,同时确保信息传递不中断。

![White List Numbers](images/whitelist1.png)

---

## 将数字添加到白名单

1. 导航到 **白色列表数字** 区域。
2. 点击 **添加新内容**。 。 。 。
3. 输入输入字段中的移动数字。
4. 通过用逗号将它们分开,可以同时添加多个数字(<span data-ph="0"></span>) (中文(简体) ).
5. 确保 **每个手机号码包括国家代码** (如:+91XXXXXXXXXX.
6. 保存更改以在全球应用白名单条目 。

![Add Whitelist Numbers](images/whitelist2.png)

---

## 删除列入白名单的数字

应用程序支持 **批量删除**,允许管理员删除单个动作中的多个白名单条目。

管理员也可删除 **单个选定条目** 视需要而定。

![Bulk Delete Whitelist](images/whitelist4.png)

---

## 关键说明

!!! info "重要"
    - 采用白名单编号 **整个应用系统**。 。 。 。
    - 发送到这些数字的信件是 **从德国航天中心赔偿计算中排除**。 。 。 。
    - 正确功能必须采用国家代码的适当格式。
