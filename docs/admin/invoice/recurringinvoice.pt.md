
## Fatura recorrente

Esta página permite- lhe **adicionar, ver e editar** os ciclos de fatura para o seu **Utilizadores pós-pagos**. 
O painel exibe todos os ciclos de fatura ativa juntamente com seus detalhes. 
Podes. **iniciar ou parar** qualquer ciclo de fatura usando o menu Ação.

### 1. Criar Fatura Recorrente
Esta opção é usada para criar um novo ciclo de faturas recorrentes para o seu usuário pós-pago. 
Siga os passos abaixo para criar um ciclo:

1. **Selecione o usuário.**
2. **Selecione o intervalo de fatura** – Semanal, quinzenal, mensal ou trimestral.
3. **Adiante em:** 
 Define o número de dias após a criação da fatura quando a fatura será considerada atrasada.
4. **Data de início e data de fim:** 
 Estas datas definem o período de validade do ciclo da factura. 
   - Se activar **Nunca Expirar**, as faturas continuarão a gerar indefinidamente.
5. **É Activo:** 
 Active esta opção para iniciar o ciclo de facturas do utilizador seleccionado.
