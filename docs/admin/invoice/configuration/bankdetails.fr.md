
### Configurer les détails bancaires

Les **Configurer les détails bancaires** section vous permet de stocker et de gérer les informations de compte bancaire où les paiements des clients devraient être effectués. 
Vous pouvez saisir des données bancaires complètes pour assurer un traitement des paiements en douceur.

Les détails suivants doivent être fournis:

1. **Nom de la branche** – Entrez le nom de la succursale bancaire.
2. **Code rapide** – Fournir le code SWIFT/BIC pour les transactions internationales.
3. **Adresse** – Ajouter l'adresse complète de la succursale bancaire.
4. **Note spéciale** – Inclure les instructions ou notes supplémentaires relatives au processus de paiement.
