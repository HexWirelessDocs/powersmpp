
## Configurations

### Paramètres généraux

Les **Paramètres généraux** section vous permet de configurer et de personnaliser les informations par défaut qui apparaissent sur toutes les factures générées. 
À partir de ce panneau, vous pouvez mettre à jour :

- **Note du client** – Une note personnalisable qui apparaîtra sur la facture pour vos utilisateurs.
- **Conditions générales** – Les termes qui seront inclus au bas de chaque facture.

Ces paramètres aident à maintenir l'uniformité de toutes les factures envoyées à vos clients.
