
### Configurar os Detalhes do Banco

A **Configurar os Detalhes do Banco** a seção permite armazenar e gerenciar as informações da conta bancária onde os pagamentos dos clientes devem ser feitos. 
Você pode inserir detalhes completos do banco para garantir o processamento de pagamentos suaves.

Devem ser fornecidos os seguintes pormenores:

1. **Nome do ramo** – Digite o nome da sucursal bancária.
2. **Código Rápido** – Forneça o código SWIFT/BIC para transações internacionais.
3. **Endereço** – Adicione o endereço completo da sucursal bancária.
4. **Nota especial** – Inclua quaisquer instruções adicionais ou notas relacionadas com o processo de pagamento.
