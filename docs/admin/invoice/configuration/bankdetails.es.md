
### Configure Bank Details

El **Configure Bank Details** sección le permite almacenar y gestionar la información de la cuenta bancaria donde se deben hacer pagos de clientes. 
Puede introducir detalles bancarios completos para asegurar el procesamiento de pago liso.

Deben proporcionarse los siguientes datos:

1. **Nombre de la rama** – Introduzca el nombre de la rama bancaria.
2. **Swift Code** – Proporcionar el código SWIFT/BIC para transacciones internacionales.
3. **Dirección** – Agregue la dirección completa de la rama bancaria.
4. **Nota especial** – Incluya instrucciones o notas adicionales relacionadas con el proceso de pago.
