
### Gérer les détails fiscaux

Les **Gérer les détails fiscaux** section vous permet de configurer les taxes applicables à votre entreprise et de les appliquer automatiquement à vos factures utilisateurs. 
Pour ajouter une nouvelle taxe, cliquez sur **Ajouter une nouvelle taxe** et précisez:

1. **Dénomination fiscale** – Un nom convivial pour la taxe.
2. **Pourcentage des impôts** – Le pourcentage de l'impôt qui doit être appliqué.
3. **Désignation des marchandises** – Une brève explication ou des détails concernant la taxe.
4. **Impôt applicable** – Activer cette option pour s'assurer que la taxe est appliquée sur toutes les factures.
