
## Gérer les paiements reçus

Cette section vous permet de **et réclamations** dès qu'ils sont reçus du client. 
Cliquez sur **Ajouter un nouveau paiement** pour enregistrer un paiement.

### Étapes pour enregistrer un paiement

1. **Sélectionnez l'utilisateur.**
2. **Sélectionnez le numéro de facture** contre lequel le paiement a été reçu.
3. **Saisissez l'identifiant de la transaction** et **ID de référence bancaire** liés au paiement.
4. **Sélectionnez le type de paiement** — en ligne, chèque, DD, etc.
5. **Entrez la date de paiement.**
6. **Inscrivez le montant** reçu du client.
