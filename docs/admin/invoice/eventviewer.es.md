
## Event Viewer (Invocando)

El **iTextPro Event Viewer (Invocando)** proporciona información detallada sobre cualquier **errores o problemas** encontrado durante el proceso de creación de facturas o durante el envío de notificaciones de facturas en el sistema.

Este módulo captura **Registros completos de eventos**, ayudar a los administradores a identificar rápidamente, analizar y resolver problemas. 
Garantiza una solución de problemas eficiente y soporta un flujo de trabajo de facturación más suave y fiable.
