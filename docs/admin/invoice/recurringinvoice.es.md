
## Factura recurrente

Esta página te permite **añadir, ver y editar** los ciclos de facturas para su **Usuarios postales**. 
El panel muestra todos los ciclos de facturas activos junto con sus detalles. 
Puedes **iniciar o parar** cualquier ciclo de facturas usando el menú Acción.

### 1. Crear facturas recurrentes
Esta opción se utiliza para crear un nuevo ciclo de facturas recurrente para su usuario Postpaid. 
Siga los pasos a continuación para crear un ciclo:

1. **Seleccione el usuario.**
2. **Seleccione el intervalo de factura** – Semanal, quincenal, mensual o trimestral.
3. **Due in:** 
 Define el número de días después de la creación de facturas cuando la factura será considerada atrasada.
4. **Fecha de inicio " Fecha final:** 
 Estas fechas definen el período de validez del ciclo de facturas. 
   - Si lo habilitas **"Never Expire"**, las facturas continuarán generando indefinidamente.
5. **Está activo:** 
 Permitir esta opción para iniciar el ciclo de facturas para el usuario seleccionado.
