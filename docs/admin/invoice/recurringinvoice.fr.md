
## Facture récurrente

Cette page vous permet de **ajouter, afficher et modifier** les cycles de facturation pour votre **Utilisateurs postpayés**. 
Le tableau de bord affiche tous les cycles de facturation actifs ainsi que leurs détails. 
Vous pouvez **démarrage ou arrêt** tout cycle de facturation utilisant le menu Action.

### 1. Créer une facture récurrente
Cette option est utilisée pour créer un nouveau cycle de facturation récurrent pour votre utilisateur Postpayé. 
Suivez les étapes ci-dessous pour créer un cycle :

1. **Sélectionnez l'utilisateur.**
2. **Sélectionnez l'intervalle de facturation** – hebdomadaire, bimensuel, mensuel ou trimestriel.
3. **Date d'échéance :** 
 Définit le nombre de jours après la création de la facture lorsque la facture sera considérée comme en retard.
4. **Date de début et de fin :** 
 Ces dates définissent la période de validité du cycle de facturation. 
   - Si vous activez **"Ne jamais expirer"**, les factures continueront à générer indéfiniment.
5. **Est actif:** 
 Activer cette option pour démarrer le cycle de facturation pour l'utilisateur sélectionné.
