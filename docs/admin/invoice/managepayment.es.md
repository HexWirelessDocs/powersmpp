
## Administrar el pago recibido

Esta sección le permite **y pagos de reclamaciones** tan pronto como sean recibidos del cliente. 
Haga clic en **“Agregar nuevo pago”** para registrar un pago.

### Pasos para registrar un pago

1. **Seleccione el Usuario.**
2. **Seleccione el número de facturación** contra el cual se ha recibido el pago.
3. **Introduzca el ID de transacción** y **ID de referencia del Banco** relacionado con el pago.
4. **Seleccione el tipo de pago** - En línea, Cheque, DD, etc.
5. **Introduzca la fecha de pago.**
6. **Introduzca el importe** recibido del cliente.
