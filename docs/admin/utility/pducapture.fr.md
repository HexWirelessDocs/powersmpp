
# Capture de PDU

Les **Capture de PDU** option est un outil puissant pour capturer **trafic de messages en direct** entrée ou sortie **Centre de service des messages courts (SMSC)**. 
Il est spécialement conçu pour **dépannage en temps réel** de la passerelle ou **Entité externe à message court (ESME)** utilisateurs.

## Détails de configuration

Pour lancer la capture de PDU, fournir les renseignements suivants :

- **Hébergement** 
 Préciser **adresse de l'hôte** ou **adresse IP du serveur** de l'interface Ethernet connectée au serveur. 
 Ceci identifie le chemin réseau pour capturer les PDU.

- **Filtre** 
 Choisissez le produit désiré **filtre** pour commencer à capturer des PDU en direct. 
 Par exemple, sélectionner **numéro de port 8585** saisira les PDU pour ce port spécifique seulement. 
 Consultez la documentation pour obtenir des lignes directrices détaillées sur la capture de PDU avec **port 8585**.

- **Intervalle** 
 Entrez l'intervalle de temps souhaité (en minutes), avec **limite maximale de 90 minutes**. 
 Cela détermine la durée de capture des PDU en direct.

---

## Remarque
Les **Capture de PDU** option est un outil de diagnostic inestimable pour **enquête en temps réel**. 
Elle est particulièrement utile pour les utilisateurs qui n'ont pas accès à une équipe de CNO ou qui possèdent des compétences avancées en dépannage réseau. 

Veiller à ce que **coordonnées de l'hôte**, **filtres**et **intervalles de temps** sont configurés avec précision pour des résultats optimaux.
