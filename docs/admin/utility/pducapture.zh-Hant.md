
# PDU 捕獲

這個 **PDU 捕獲** 選項是一個強大的工具,用於抓取 **實時資訊傳輸** 進入或退出 **簡訊息服務中心( SMSC)**。 。 。 。 
它是專門為 **實時排除故障** 閘道器或 **外部短訊息實體( ESME)** 使用者。

## 配置細節

為了啟動PDU捕獲,提供以下資訊:

- **東道主** 
 指定 **主機地址** 或者說 **伺服器 IP 地址** 連線到伺服器的乙太網介面。 
 這可以識別捕獲PDU的網路路徑.

- **過濾器** 
 選擇想要的 **過濾器** 開始捕捉活的PDU。 
 例如,選擇 **8585號埠** 將只捕捉該特定埠的PDU。 
 關於PDU捕獲的詳細準則,請參考檔案。 **埠8585**。 。 。 。

- **時間間隔** 
 輸入想要的時間間隔( 以分鐘計), 有 **最長限為90分鐘**。 。 。 。 
 這決定了活下來的PDU被俘虜的時間.

---

## 說明
這個 **PDU 捕獲** 選項是用於 **實時問題調查**。 。 。 。 
這對於可能無法進入NOC團隊或擁有高階網路故障排除技能的使用者特別有用. 

確保 **主機細節**, (中文(簡體) ). **過濾器**,以及 **時間間隔** 為最佳結果而精確配置。
