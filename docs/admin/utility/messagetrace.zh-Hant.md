
# 信件追蹤

這個 **信件追蹤** 選項在 iTextPro 中授權您 **跟蹤和分析訊息報告** 精確地說 
透過使用可用的過濾器,您可以檢索關於特定訊息的詳細資訊,從而更容易調查和解決問題.

---

## 可用過濾器

### 1. 發件人身份證
基於 **傳送者身份證明**。 。 。 。 
這有助於您關注特定傳送者傳送的資訊 。

### 2. 移動號碼
搜尋一個 **特定移動號碼** 檢索與該收件人有關的資訊的詳細報告。

### 3. 資訊內容
使用 **關鍵詞或確切內容** 查詢和分析基於其文字的資訊。

---

## 使用大小寫
這一特點在解決特別報告員提出的關切時特別重要。 **電信供應商**,例如:
- 不干擾( N)**德國**() 違反
- **黑名單號碼**
- **潛在的垃圾郵件** 交通

透過使用這些過濾器追蹤訊息,您可以快速調查並回應任何已報告的差異.

---

## 重要說明
這個 **信件追蹤** 選項允許您審查信件報告 **僅過去3天**。 。 。 。 
在解決問題或解決問題時牢記這一限制。
