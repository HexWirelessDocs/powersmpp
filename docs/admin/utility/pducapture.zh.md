
# PDU 捕获

这个 **PDU 捕获** 选项是一个强大的工具,用于抓取 **实时信息传输** 进入或退出 **短信息服务中心( SMSC)**。 。 。 。 
它是专门为 **实时排除故障** 网关或 **外部短消息实体( ESME)** 用户。

## 配置细节

为了启动PDU捕获,提供以下信息:

- **东道主** 
 指定 **主机地址** 或者说 **服务器 IP 地址** 连接到服务器的以太网接口。 
 这可以识别捕获PDU的网络路径.

- **过滤器** 
 选择想要的 **过滤器** 开始捕捉活的PDU。 
 例如,选择 **8585号端口** 将只捕捉该特定端口的PDU。 
 关于PDU捕获的详细准则,请参考文件。 **端口8585**。 。 。 。

- **时间间隔** 
 输入想要的时间间隔( 以分钟计), 有 **最长限为90分钟**。 。 。 。 
 这决定了活下来的PDU被俘虏的时间.

---

## 说明
这个 **PDU 捕获** 选项是用于 **实时问题调查**。 。 。 。 
这对于可能无法进入NOC团队或拥有高级网络故障排除技能的用户特别有用. 

确保 **主机细节**, (中文(简体) ). **过滤器**,以及 **时间间隔** 为最佳结果而精确配置。
