
# 查看流量

这个 **查看流量** 选项提供细目 **网关性能** 显示不同时间间隔的吞吐量度。 
它有助于监测信息处理率和确定性能模式。

---

## 使用大小写
这一特点对以下方面来说是宝贵的:
- **评估网关效率**
- 跟踪通过量趋势 **实时** 或超过 **历史时期**
- 识别 **高峰装载期**
- 寻找机会 **资源优化**

---

## 如何使用
1. **选择想要的时间过滤器** 以显示相应的吞吐量数据。
2. 评议 **视觉表现** (a) 产出计量。
3. 确定模式、趋势和潜在优化领域。

通过利用所提供的见解,你可以 **数据驱动的决定** 改进业绩和加强系统监测。
