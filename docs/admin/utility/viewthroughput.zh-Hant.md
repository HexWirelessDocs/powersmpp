
# 檢視流量

這個 **檢視流量** 選項提供細目 **閘道器效能** 顯示不同時間間隔的吞吐量度。 
它有助於監測資訊處理率和確定性能模式。

---

## 使用大小寫
這一特點對以下方面來說是寶貴的:
- **評估閘道器效率**
- 跟蹤透過量趨勢 **實時** 或超過 **歷史時期**
- 識別 **高峰裝載期**
- 尋找機會 **資源最佳化**

---

## 如何使用
1. **選擇想要的時間過濾器** 以顯示相應的吞吐量資料。
2. 評議 **視覺表現** (a) 產出計量。
3. 確定模式、趨勢和潛在最佳化領域。

透過利用所提供的見解,你可以 **資料驅動的決定** 改進業績和加強系統監測。
