
# 工具 — 队列到队列传输

这个 **队列到队列传输** iTextPro 中的特性使网关之间的短信流量能够无缝地传输,确保信息传送业务的有效管理。 
此功能在您 **主网关** 体验停机时间或变得不起作用。 通过这个选项,你可以 **将流量重定向到另一个网关**确保不间断的通信。

![Queue-to-Queue Overview](../images/queue1.png)

---

## 传输配置

### 1. 转让目的
指定传输短信的目的或原因。 
帮助维持 **清晰的记录** 并提供转让的背景。

### 2. 来源网关
选择 **源网关** 将从其中传送信件。 
iTextPro 允许您选择 **特定消息队列** 与此网关相关。

### 3. 目标网关
选择 **目的地网关** (c) 应在何处发送信息。 
这确保了交通改道到正确的运行终点.

### 4. 启动转让
一旦设定了传输参数:
- 点击 **"添加任务"** 按钮。
- iTextPro 将把任务添加到传输队列中.
- 系统将自动传输所选信件 **源网关** 页:1 **目的地网关**。 。 。 。

这种无缝的流程保证 **短信服务的连续性** 即使是在网关关闭的情况下。
