
# 工具 — 佇列到佇列傳輸

這個 **佇列到佇列傳輸** iTextPro 中的特性使閘道器之間的簡訊流量能夠無縫地傳輸,確保資訊傳送業務的有效管理。 
此功能在您 **主閘道器** 體驗停機時間或變得不起作用。 透過這個選項,你可以 **將流量重定向到另一個閘道器**確保不間斷的通訊。

![Queue-to-Queue Overview](../images/queue1.png)

---

## 傳輸配置

### 1. 轉讓目的
指定傳輸簡訊的目的或原因。 
幫助維持 **清晰的記錄** 並提供轉讓的背景。

### 2. 來源閘道器
選擇 **源閘道器** 將從其中傳送信件。 
iTextPro 允許您選擇 **特定訊息佇列** 與此閘道器相關。

### 3. 目標閘道器
選擇 **目的地閘道器** (c) 應在何處傳送資訊。 
這確保了交通改道到正確的執行終點.

### 4. 啟動轉讓
一旦設定了傳輸引數:
- 點選 **"新增任務"** 按鈕。
- iTextPro 將把任務新增到傳輸佇列中.
- 系統將自動傳輸所選信件 **源閘道器** 頁:1 **目的地閘道器**。 。 。 。

這種無縫的流程保證 **簡訊服務的連續性** 即使是在閘道器關閉的情況下。
